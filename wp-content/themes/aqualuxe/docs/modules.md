# AquaLuxe Module Reference

## Wholesale
- Location: `modules/wholesale/`
- Provides wholesale pricing and registration features.
- Custom templates: `templates/wholesale-notice.php`
- Service: `inc/core/services/WholesaleService.php`

## Classifieds
- Location: `modules/classifieds/`
- Enables classified ads and listings.
- Custom templates: `templates/ad-summary.php`
- Service: `inc/core/services/ClassifiedsService.php`

## Trade-Ins
- Location: `modules/trade-ins/`
- Allows users to submit trade-in forms.
- Custom templates: `templates/trade-in-form.php`
- Service: `inc/core/services/TradeInService.php`

## Services
- Location: `modules/services/`
- Manages service listings and bookings.
- Custom templates: `templates/service-summary.php`, `templates/service-single.php`
- Service: `inc/core/services/ConsultationService.php`

## Franchise
- Location: `modules/franchise/`
- Franchise directory and application forms.
- Custom templates: `templates/franchise-summary.php`, `templates/franchise-archive.php`, `templates/franchise-single.php`
- Service: `inc/core/services/FranchiseService.php`

## Sustainability
- Location: `modules/sustainability/`
- Sustainability initiatives and content.
- Custom templates: `templates/sustainability-summary.php`, `templates/sustainability-archive.php`, `templates/sustainability-single.php`
- Service: `inc/core/services/SustainabilityService.php`

## Affiliate
- Location: `modules/affiliate/`
- Affiliate dashboard and tracking.
- Custom templates: `templates/affiliate-dashboard.php`
- Service: `inc/core/services/AffiliateService.php`

## Multitenancy
- Location: `modules/multitenancy/`
- Multi-tenant site features.
- Custom templates: `templates/tenant-banner.php`
- Service: `inc/core/services/MultitenancyService.php`

---

Each module can be extended or customized independently. See the module's `README.md` for more details.
