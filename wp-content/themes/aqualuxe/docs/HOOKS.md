# AquaLuxe Hooks & Filters

- `aqualuxe_enabled_modules`: array of module folder names to load.
- `aqualuxe_classifieds_cpt_args`: filter CPT args for classifieds.
- `aqualuxe_classifieds_category_tax_args`: filter category taxonomy args.
- `aqualuxe_classifieds_location_tax_args`: filter location taxonomy args.
- `aqualuxe_classifieds_default_status`: filter default submission status.
- `aqualuxe_classifieds_submitted`: action after a submission.
- `aqualuxe_classifieds_approved_subject`/`_body`: email content filters.
