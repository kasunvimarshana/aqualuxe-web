# Install & Setup

1. Copy `aqualuxe` into `wp-content/themes/`.
2. In `wp-content/themes/aqualuxe`:
   - `npm install`
   - `npm run build`
3. Activate the theme in WP Admin.
4. (Optional) Tools → AquaLuxe Importer to seed demo content.

Notes:
- Requires PHP 8.0+, WP 6.3+, WooCommerce optional.
- No external CDNs; assets are built into `assets/dist`.
