# Modules

Each feature lives under `modules/<feature>/` with a `bootstrap.php` and may add hooks, shortcodes, UI, and templates.

Enabled by default: dark_mode, multilingual, importer, wishlist, quick_view, advanced_filters, multicurrency, roles, marketplace, classifieds.

Toggle via:
- Option `aqualuxe_enabled_modules_option` (array of folder names)
- Filter `aqualuxe_enabled_modules`
