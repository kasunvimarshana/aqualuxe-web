## Accessing the Customizer

To access the WordPress Customizer:

1. Log in to your WordPress admin dashboard
2. Navigate to **Appearance > Customize**
3. The Customizer will open with a live preview of your site
4. Make changes and see them in real-time before publishing

## Site Identity

### Logo

1. In the Customizer, click on **Site Identity**
2. Click **Select logo** to upload your logo
3. Recommended logo size: 250px × 100px
4. For Retina displays, upload a logo twice the display size
5. Adjust the logo size using the slider if needed
6. Click **Publish** to save changes

### Site Title and Tagline

1. In the Customizer, click on **Site Identity**
2. Enter your site title and tagline
3. Toggle the display of the site title and tagline
4. Click **Publish** to save changes

### Site Icon (Favicon)

1. In the Customizer, click on **Site Identity**
2. Click **Select site icon** to upload your favicon
3. Recommended size: 512px × 512px
4. The icon will be resized automatically for different devices
5. Click **Publish** to save changes

## General Settings

### Layout Options

1. In the Customizer, click on **General Settings > Layout Options**
2. Choose between full-width or boxed layout
3. Set the content width (default: 1200px)
4. Configure sidebar options:
   - Left sidebar
   - Right sidebar
   - No sidebar
5. Click **Publish** to save changes

### Background

1. In the Customizer, click on **General Settings > Background**
2. Set the background color
3. Or upload a background image
4. Configure background image options:
   - Background repeat
   - Background position
   - Background attachment
   - Background size
5. Click **Publish** to save changes

### Buttons

1. In the Customizer, click on **General Settings > Buttons**
2. Customize button appearance:
   - Button border radius
   - Button padding
   - Button text transform
3. Click **Publish** to save changes

## Custom CSS

For advanced customization, you can add custom CSS:

1. In the Customizer, click on **Additional CSS**
2. Add your custom CSS code
3. See the changes in real-time
4. Click **Publish** to save changes

Example custom CSS:

```css
/* Change the primary button hover color */
.button.button-primary:hover {
    background-color: #0056b3;
}

/* Increase the font size of product titles */
.woocommerce ul.products li.product .woocommerce-loop-product__title {
    font-size: 18px;
}
```

## Using a Child Theme

For extensive customizations, it's recommended to use a child theme:

1. Create a new folder in your themes directory named `aqualuxe-child`
2. Create a `style.css` file with the following content:

```css
/*
Theme Name: AquaLuxe Child
Theme URI: https://example.com/aqualuxe-child/
Description: A child theme of AquaLuxe
Author: Your Name
Author URI: https://example.com
Template: aqualuxe
Version: 1.0.0
*/

/* Import parent theme style */
@import url("../aqualuxe/style.css");

/* Add your custom styles below this line */
```

3. Create a `functions.php` file with the following content:

```php
<?php
/**
 * AquaLuxe Child Theme functions and definitions
 */

// Enqueue parent and child theme styles
function aqualuxe_child_enqueue_styles() {
    wp_enqueue_style('aqualuxe-style', get_template_directory_uri() . '/style.css');
    wp_enqueue_style('aqualuxe-child-style', get_stylesheet_directory_uri() . '/style.css', array('aqualuxe-style'));
}
add_action('wp_enqueue_scripts', 'aqualuxe_child_enqueue_styles');

// Add your custom functions below this line
```

4. Activate the child theme from **Appearance > Themes**

## Advanced Customization

### Template Files

To customize template files:

1. Create a folder with the same name as the parent theme template folder in your child theme
2. Copy the template file from the parent theme to the child theme
3. Edit the file in the child theme

For example, to customize the product page:

1. Copy `woocommerce/single-product.php` from the parent theme to your child theme's `woocommerce` folder
2. Edit the file in your child theme

### Hooks and Filters

AquaLuxe provides various hooks and filters for customization:

```php
// Add content before the header
function my_custom_before_header() {
    echo '<div class="announcement-bar">Special offer: Free shipping on orders over $50!</div>';
}
add_action('aqualuxe_before_header', 'my_custom_before_header');

// Modify the footer copyright text
function my_custom_copyright_text($text) {
    return 'Copyright © ' . date('Y') . ' My Aquarium Shop. All rights reserved.';
}
add_filter('aqualuxe_copyright_text', 'my_custom_copyright_text');
```

### Custom Page Templates

To create a custom page template:

1. Create a new file in your child theme
2. Add the template header:

```php
<?php
/**
 * Template Name: My Custom Template
 */

get_header();
?>

<!-- Your custom template content here -->

<?php
get_footer();
```

3. The template will be available in the Page Attributes section when editing a page
