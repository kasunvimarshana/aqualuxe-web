# AquaLuxe Pro Theme

## Overview

AquaLuxe Pro is a modern, modular, and highly customizable WordPress theme designed for e-commerce, with a special focus on ornamental fish and aquarium supplies. It's built with a dual-state architecture to work seamlessly with or without WooCommerce.

## Features

*   **Modular Architecture:** Core functionalities are separated from feature modules, making the theme extensible and maintainable.
*   **WooCommerce Ready:** Deep integration with WooCommerce, providing a seamless shopping experience.
*   **Theme Customizer:** A wide range of options to customize the theme's appearance, including colors, typography, and layout.
*   **Dark Mode:** A built-in dark mode switcher for a better user experience in low-light environments.
*   **Multivendor Support:** Compatibility with Dokan to create a multivendor marketplace.
*   **Multilingual and Multicurrency Ready:** Placeholders and hooks for easy integration with multilingual and multicurrency plugins.
*   **Bookings Ready:** Compatibility with WooCommerce Bookings for service-based businesses.
*   **Responsive Design:** Mobile-first design that looks great on all devices.
*   **Demo Content Importer:** Easily import demo content to get your site up and running quickly.

## Installation

1.  Download the theme from the official repository.
2.  In your WordPress admin panel, go to **Appearance > Themes** and click **Add New**.
3.  Click **Upload Theme** and choose the downloaded zip file.
4.  Click **Install Now** and then **Activate**.

For more detailed instructions, see the [Installation Guide](docs/installation.md).

## Configuration

After installation, you can configure the theme through the WordPress Customizer, available under **Appearance > Customize**. Here you can set up your logo, colors, fonts, and other theme options.

## Developer Documentation

For developers, the theme provides a clean and organized codebase with a modular structure. All assets are compiled using Laravel Mix. See the [Developer Guide](docs/developer-guide.md) for more information on the theme's architecture, hooks, and filters.

## User Documentation

For end-users, the theme is designed to be intuitive and easy to use. The [User Guide](docs/user-guide.md) provides detailed instructions on how to use the theme's features.
