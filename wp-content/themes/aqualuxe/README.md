# AquaLuxe - Premium WooCommerce Child Theme

A premium WooCommerce child theme for ornamental fish businesses, built on the Storefront parent theme.

## Description

AquaLuxe is a premium WooCommerce child theme designed specifically for ornamental fish businesses. Built on the robust Storefront parent theme, it provides a beautiful, responsive, and fully customizable e-commerce solution.

## Features

- **Fully Responsive Design**: Looks great on all devices
- **WooCommerce Integration**: Complete compatibility with all WooCommerce features
- **Customizer Options**: Easy branding and customization
- **SEO Optimized**: Built with search engines in mind
- **Accessibility Ready**: WCAG 2.0 compliant
- **Performance Optimized**: Fast loading times
- **Demo Content**: One-click demo import
- **AJAX Add to Cart**: Enhanced shopping experience
- **Product Quick View**: Preview products without leaving the page

## Installation

1. Install and activate the Storefront parent theme
2. Upload the AquaLuxe theme folder to your `/wp-content/themes/` directory
3. Activate the AquaLuxe theme from your WordPress dashboard
4. Go to "Appearance > Customize" to start customizing your theme

## Customization

AquaLuxe offers extensive customization options through the WordPress Customizer:

- Color schemes
- Typography
- Header options
- Layout settings
