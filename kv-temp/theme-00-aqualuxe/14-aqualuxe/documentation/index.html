Introduction

Welcome to the documentation for AquaLuxe, a premium WordPress theme designed specifically for high-end ornamental fish farming businesses. This theme combines elegant aquatic visuals with refined typography and smooth micro-interactions to create a luxurious online experience.

AquaLuxe features full WooCommerce integration with all standard e-commerce functionality, making it perfect for selling rare fish species, aquarium plants, equipment, and related services.

Tip: Always backup your site before making significant changes or updates.

Installation

Prerequisites

  • WordPress 5.0 or higher
  • PHP 7.0 or higher
  • MySQL 5.6 or higher
  • WooCommerce plugin (recommended)

Installing the Theme

  1. Download the AquaLuxe theme package
  2. Log in to your WordPress admin dashboard
  3. Navigate to Appearance > Themes
  4. Click Add New then Upload Theme
  5. Choose the downloaded theme file and click Install
  6. Click Activate to activate the theme

Initial Setup

Setting Up Menus

  1. Go to Appearance > Menus
  2. Create a new menu or edit an existing one
  3. Add pages, categories, or custom links to your menu
  4. Assign the menu to a theme location (Primary Menu or Footer Menu)
  5. Save the menu

Setting Up Widgets

  1. Go to Appearance > Widgets
  2. Drag and drop widgets to the available widget areas (Sidebar, Footer 1-4)
  3. Configure each widget as needed
  4. Save changes

Theme Customizer

The Theme Customizer allows you to make changes to your site and see a preview of those changes in real-time.

Accessing the Customizer

  1. Go to Appearance > Customize
  2. Use the panels on the left to navigate through customization options
  3. See your changes in the preview pane on the right
  4. Click Publish to save your changes

Customization Options

WooCommerce Setup

Installing WooCommerce

  1. Go to Plugins > Add New
  2. Search for "WooCommerce"
  3. Click Install Now and then Activate
  4. Follow the WooCommerce setup wizard to configure your store

Configuring Payment Gateways

  1. Go to WooCommerce > Settings > Payments
  2. Enable and configure payment methods (PayPal, Stripe, etc.)
  3. Test payment processing

Setting Up Shipping

  1. Go to WooCommerce > Settings > Shipping
  2. Configure shipping zones and methods
  3. Set up shipping classes if needed

Demo Content

AquaLuxe includes demo content to help you get started quickly.

Importing Demo Content

  1. Go to Appearance > Import Demo Data
  2. Choose a demo to import (Main Demo or Minimal Demo)
  3. Click Import Demo
  4. Wait for the import process to complete
  5. Review and customize the imported content
Warning: Importing demo content will overwrite your existing content. Always backup your site before importing.

Customization

Child Theme

We recommend using a child theme for any customizations to ensure your changes aren't lost during theme updates.

Custom CSS

You can add custom CSS through the Customizer:

  1. Go to Appearance > Customize > Additional CSS
  2. Add your custom CSS
  3. Click Publish

Custom Hooks

The theme provides several hooks for developers:

  • aqualuxe_before_header - Before the header
  • aqualuxe_after_header - After the header
  • aqualuxe_before_content - Before the main content
  • aqualuxe_after_content - After the main content
  • aqualuxe_before_footer - Before the footer
  • aqualuxe_after_footer - After the footer

Troubleshooting

Common Issues

  • Missing styles: Ensure all theme files are uploaded correctly and clear any caching plugins.
  • WooCommerce pages not displaying correctly: Make sure WooCommerce is installed and activated, and check that the correct pages are assigned in WooCommerce settings.
  • Menu not displaying: Check that a menu is assigned to the Primary Menu location in Appearance > Menus.

Getting Support

If you're experiencing issues not covered in this documentation, please contact our support team with the following information:

  • WordPress version
  • Theme version
  • List of active plugins
  • Description of the issue
