# AquaLuxe Installation Guide

## Quick Installation

1. **Install Storefront Parent Theme**
   - Go to Appearance > Themes in your WordPress admin
   - Click "Add New"
   - Search for "Storefront"
   - Install and activate the Storefront theme

2. **Install AquaLuxe Child Theme**
   - Download the AquaLuxe theme files
   - Go to Appearance > Themes in your WordPress admin
   - Click "Add New"
   - Click "Upload Theme"
   - Choose the AquaLuxe theme ZIP file
   - Click "Install Now"
   - Click "Activate"

3. **Install WooCommerce (if not already installed)**
   - Go to Plugins > Add New
   - Search for "WooCommerce"
   - Install and activate WooCommerce
   - Follow the setup wizard to configure your store

4. **Configure Theme Settings**
   - Go to Appearance > Customize
   - Adjust colors, typography, and layout settings
   - Upload your logo
   - Configure menu locations

5. **Import Demo Content (Optional)**
   - Go to Appearance > Demo Content
   - Click "Import Demo Content"
   - Confirm the import when prompted

## System Requirements

- WordPress 5.8 or higher
- PHP 7.4 or higher
- WooCommerce 6.0 or higher
- Storefront theme (parent theme)

## Troubleshooting

### Common Issues

1. **Theme not activating**
   - Ensure Storefront parent theme is installed and activated
   - Check that your WordPress version meets the minimum requirements

2. **Missing WooCommerce features**
   - Verify WooCommerce is installed and activated
   - Check that WooCommerce version meets the minimum requirements

3. **Demo content not importing**
   - Ensure you have administrator privileges
   - Check that your server meets the minimum requirements for file uploads
