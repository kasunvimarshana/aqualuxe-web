A premium WooCommerce child theme for ornamental fish businesses.

== Description ==

AquaLuxe is a premium WooCommerce child theme built on the official Storefront parent theme. It's specifically designed for ornamental fish businesses targeting both local and international markets. The theme follows WordPress and WooCommerce coding standards with SOLID, DRY, and KISS principles for maximum readability, reusability, extendibility, customizability, scalability, and maintainability.

Key features:
* Fully responsive design that adapts to different screen sizes and devices
* Clean and modern layout that showcases products beautifully
* Full WooCommerce compatibility (shop, cart, checkout, account, etc.)
* Support for variable products, grouped products, and digital/physical goods
* AJAX add-to-cart functionality
* Product quick view feature
* Sticky headers and mobile-friendly navigation
* Custom widget areas
* SEO optimization with schema markup and Open Graph metadata
* Accessibility features with ARIA roles
* Lazy loading for images
* Demo content import system

== Installation ==

1. Install and activate the Storefront parent theme
2. Upload the AquaLuxe theme folder to your WordPress themes directory (`/wp-content/themes/`)
3. Activate the AquaLuxe theme from the WordPress admin panel
4. Navigate to "Appearance > Customize" to configure theme options
5. (Optional) Navigate to "Appearance > Import Demo Data" to import sample content
