# AquaLuxe WooCommerce Child Theme

## Overview
AquaLuxe is a premium WooCommerce child theme built on the official Storefront parent theme. Designed specifically for ornamental fish businesses, it combines elegance with functionality to create a stunning online store that showcases aquatic products beautifully.

## Installation

### Method 1: WordPress Admin (Recommended)
1. Install and activate the Storefront parent theme
2. Navigate to **Appearance > Themes > Add New**
3. Click **Upload Theme**
4. Select the `aqualuxe.zip` file
5. Click **Install Now**
6. Click **Activate**

### Method 2: FTP Upload
1. Extract the `aqualuxe.zip` file
2. Upload the `aqualuxe` folder to `/wp-content/themes/`
3. Navigate to **Appearance > Themes**
4. Activate the AquaLuxe theme

### Initial Setup
1. Run the WooCommerce setup wizard
2. Import demo content (optional)
3. Customize theme options via **Appearance > Customize**
4. Configure menus and widgets
5. Add your products

## Frequently Asked Questions

### Do I need the Storefront theme?
Yes, Storefront is required as the parent theme for AquaLuxe to function properly.

### Can I use this theme without WooCommerce?
While possible, the theme is specifically designed for e-commerce and many features will be underutilized.

### How do I update the theme?
1. Backup your website
2. Download the latest version
3. Upload via WordPress admin or FTP
4. Test your site after update

### Will my customizations be lost during updates?
If you use a child theme or the WordPress Customizer, your customizations will be preserved.
