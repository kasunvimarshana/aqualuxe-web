## Theme Overview

AquaLuxe is a premium WooCommerce child theme built on the official Storefront parent theme, specifically designed for ornamental fish businesses. The theme follows WordPress and WooCommerce coding standards with SOLID, DRY, and KISS principles for maximum readability, reusability, extendibility, customizability, scalability, and maintainability.

## Installation Requirements

1. WordPress 5.0 or higher
2. WooCommerce 4.0 or higher
3. Storefront parent theme (required)

## Installation Instructions

1. Install and activate the Storefront parent theme
2. Upload the `aqualuxe.zip` file to your WordPress themes directory (`/wp-content/themes/`)
3. Activate the AquaLuxe theme from the WordPress admin panel
4. Navigate to "Appearance > Customize" to configure theme options
5. (Optional) Navigate to "Appearance > Import Demo Data" to import sample content
