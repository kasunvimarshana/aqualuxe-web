A premium WooCommerce child theme for ornamental fish businesses.

== Description ==

AquaLuxe is a premium WooCommerce child theme designed specifically for ornamental fish businesses. Built on the Storefront parent theme, it provides a beautiful, responsive, and fully customizable e-commerce solution.

Key features:
* Fully responsive design
* WooCommerce integration
* Customizer options for branding
* Demo content import
* SEO optimized
* Accessibility ready
* Performance optimized

== Installation ==

1. Install and activate the Storefront theme
2. Upload the AquaLuxe theme folder to your /wp-content/themes/ directory
3. Activate the AquaLuxe theme from the WordPress admin
4. Go to Appearance > Customize to configure the theme options
