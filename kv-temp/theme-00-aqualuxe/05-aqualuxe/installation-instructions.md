# AquaLuxe Theme Installation Instructions

## Prerequisites

Before installing the AquaLuxe theme, ensure you have the following:

1. WordPress 5.0 or higher
2. PHP 7.4 or higher
3. Storefront theme (parent theme)
4. WooCommerce plugin (for e-commerce functionality)

## Installation Steps

### 1. Install Storefront Parent Theme

1. Log in to your WordPress admin dashboard
2. Go to **Appearance > Themes**
3. Click **Add New**
4. Search for "Storefront"
5. Click **Install** and then **Activate**

### 2. Install AquaLuxe Child Theme

1. Download the AquaLuxe theme files
2. Go to **Appearance > Themes** in your WordPress admin
3. Click **Add New** then **Upload Theme**
4. Choose the AquaLuxe theme .zip file
5. Click **Install Now**
6. Once installed, click **Activate**

### 3. Configure Theme Settings

1. Go to **Appearance > Customize**
2. Configure the following settings:
   - **Site Identity**: Add your logo and site title
   - **Colors**: Set your primary and secondary colors
   - **Header Media**: Add header image if desired
   - **Menus**: Create and assign menus
   - **Widgets**: Add widgets to sidebars and footer
   - **AquaLuxe Branding**: Customize theme-specific options

### 4. Import Demo Content (Optional)

1. Go to **Tools > AquaLuxe Demo Import**
2. Click **Import Demo Content**
3. Wait for the import process to complete

### 5. Configure WooCommerce Settings

1. Go to **WooCommerce > Settings**
2. Configure the following sections:
   - **General**: Store address, selling location(s)
   - **Products**: Inventory, download options
   - **Tax**: Tax options
   - **Shipping**: Shipping zones and methods
   - **Payments**: Payment gateways
   - **Accounts**: Account pages, privacy policy

### 6. Add Products

1. Go to **Products > Add New**
2. Enter product details:
   - Title
   - Description
   - Price
   - Inventory
   - Images
3. Click **Publish**

## Performance Optimization

The AquaLuxe theme includes several performance optimization features:

1. **Lazy Loading**: Images load only when they enter the viewport
2. **Minified Assets**: CSS and JavaScript files are minified
3. **DNS Prefetch**: DNS prefetching for external resources
4. **Preloading**: Critical resources are preloaded
5. **Defer JavaScript**: Non-critical JavaScript is deferred

## Troubleshooting

### Common Issues

1. **Theme not activating**
   - Ensure Storefront parent theme is installed and activated
   - Check WordPress and PHP version requirements

2. **Missing styles**
   - Ensure Storefront parent theme is activated
   - Check that all theme files are uploaded correctly

3. **WooCommerce not working**
   - Ensure WooCommerce plugin is installed and activated
   - Check WooCommerce settings

## Updating the Theme

To update the theme:

1. Backup your site
2. Download the latest version of the theme
3. Go to **Appearance > Themes**
4. Deactivate the current AquaLuxe theme
5. Delete the current AquaLuxe theme
6. Upload and install the new version
7. Activate the updated theme

Note: Customizations made directly to theme files will be lost during updates. Use a child theme or custom plugin for modifications.
