# AquaLuxe Installation Guide

## System Requirements

Before installing the AquaLuxe theme, ensure your server meets the following requirements:

- WordPress 5.0 or higher
- PHP 7.4 or higher
- MySQL 5.6 or higher
- WooCommerce 6.0 or higher

## Installation Steps

### 1. Install Storefront Parent Theme

AquaLuxe is a child theme of Storefront, so you need to install and activate Storefront first:

1. Go to **Appearance > Themes** in your WordPress admin panel
2. Click **Add New**
3. Search for "Storefront"
4. Click **Install** and then **Activate**

### 2. Install AquaLuxe Child Theme

1. Download the AquaLuxe theme files
2. Go to **Appearance > Themes** in your WordPress admin panel
3. Click **Add New** and then **Upload Theme**
4. Choose the AquaLuxe theme zip file and click **Install Now**
5. Click **Activate** to activate the theme

### 3. Configure Theme Options

1. Go to **Appearance > Customize**
2. Configure the following options:
   - **Site Identity**: Add your logo, site title, and tagline
   - **Colors**: Set your primary and secondary colors
   - **Layout**: Choose your preferred layout options
   - **Typography**: Select your preferred fonts
3. Click **Publish** to save your changes

### 4. Install Required Plugins

AquaLuxe works best with the following plugins:

- **WooCommerce**: For e-commerce functionality
- **One Click Demo Import**: For importing demo content (optional)

To install these plugins:

1. Go to **Plugins > Add New**
2. Search for the plugin name
3. Click **Install Now** and then **Activate**

### 5. Import Demo Content (Optional)

If you want to import demo content to get started quickly:

1. Install and activate the **One Click Demo Import** plugin
2. Go to **Appearance > Import Demo Data**
3. Click **Import Demo Data**
4. Follow the prompts to complete the import

### 6. Configure WooCommerce

1. Go to **WooCommerce > Settings**
2. Configure the following tabs:
   - **General**: Set your store address and selling locations
   - **Products**: Set your product settings
   - **Tax**: Configure tax settings
   - **Shipping**: Set up shipping zones and methods
   - **Payments**: Enable and configure payment gateways
3. Click **Save Changes** after configuring each tab

## Troubleshooting

### Common Issues

1. **Theme not activating**: Make sure Storefront is installed and activated before activating AquaLuxe
2. **Missing styles**: Clear your cache and ensure all plugins are up to date
3. **WooCommerce not working**: Ensure WooCommerce is installed and activated

### Support

If you encounter any issues during installation, please contact our support team at support@aqualuxe.com or visit our support forum at https://aqualuxe.com/support.

## Updating the Theme

To update AquaLuxe:

1. Download the latest version of the theme
2. Go to **Appearance > Themes** in your WordPress admin panel
3. Click **Add New** and then **Upload Theme**
4. Choose the new theme zip file
5. Click **Install Now**
6. Activate the theme if it's not already active

Note: Always backup your site before updating the theme.
