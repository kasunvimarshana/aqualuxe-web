A premium WooCommerce child theme for ornamental fish businesses.

== Description ==

AquaLuxe is a premium WooCommerce child theme designed specifically for ornamental fish businesses. Built on the Storefront parent theme, it offers a clean, modern, and responsive design that showcases your products beautifully. With full WooCommerce integration, it provides all the necessary e-commerce functionality while maintaining excellent performance and SEO optimization.

Key features include:
- Fully responsive design that works on all devices
- WooCommerce optimized for selling physical and digital products
- Support for variable and grouped products
- AJAX add-to-cart functionality
- Product quick view
- Customizer options for easy branding
- SEO optimized with schema markup and Open Graph metadata
- Secure coding practices with input sanitization and output escaping
- Demo content system for easy setup
- Extensive documentation and installation instructions

== Installation ==

1. Install and activate the Storefront parent theme
2. Upload the aqualuxe theme folder to your /wp-content/themes/ directory
3. Activate the AquaLuxe theme from the WordPress admin panel
4. Go to Appearance > Customize to configure the theme options
5. Import demo content if desired (requires One Click Demo Import plugin)
