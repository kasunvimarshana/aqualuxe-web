A premium WooCommerce child theme for ornamental fish businesses.

== Description ==

AquaLuxe is a luxurious, responsive WooCommerce child theme designed specifically for ornamental fish businesses. It features a clean, modern design that showcases the beauty of your fish while providing an excellent shopping experience for your customers.

Key features:

- Fully responsive design that works on all devices
- Custom color scheme with deep ocean blues and gold accents
- Product quick view functionality
- Sticky add-to-cart bar on product pages
- Custom product tabs for care guides
- Optimized for both local and international markets
- SEO-friendly structure
- Fast loading performance

AquaLuxe is built on the Storefront parent theme for maximum compatibility and performance.

== Installation ==

1. Upload the "aqualuxe" folder to your /wp-content/themes/ directory
2. Navigate to Appearance > Themes in your WordPress dashboard
3. Activate the AquaLuxe theme
4. Install and activate the WooCommerce plugin if you haven't already
5. Configure your settings under Appearance > Customize

Yes, AquaLuxe is designed to be compatible with most popular WooCommerce extensions.
