# AquaLuxe WooCommerce Theme

## Overview
AquaLuxe is a premium WooCommerce theme crafted for ornamental fish stores. Fully responsive, SEO-optimized, and packed with features like AJAX add-to-cart, quick view, mega menu, and more.

## Installation
1. Upload the `aqualuxe` folder to `/wp-content/themes/`  
2. Activate the theme from WordPress Admin  
3. Install and activate WooCommerce plugin  
4. Customize via Appearance > Customize

## Features
- Full WooCommerce integration  
- Responsive design  
- AJAX add-to-cart and quick view  
- Grid/List product toggle  
- Mega menu and sticky header  
- SEO schema markup  
- Performance optimized
