## Introduction

AquaLuxe provides extensive customization options through the WordPress Customizer, allowing you to tailor the appearance and functionality of your site without writing any code. This document provides a comprehensive guide to all available customization options.

## Accessing the Customizer

To access the WordPress Customizer:

1. Log in to your WordPress admin panel
2. Navigate to Appearance > Customize
3. The Customizer will open with a live preview of your site
4. Make changes and see them in real-time
5. Click "Publish" to save your changes

## Colors

The Colors section allows you to customize the color scheme of your site:

### Primary Color

The primary color is used for buttons, links, and other accent elements throughout the site.

- **Default**: #0891b2 (Cyan 600)
- **Recommendation**: Choose a color that represents your brand identity
- **Impact**: Changes the color of primary buttons, links, and accent elements

### Secondary Color

The secondary color is used for secondary buttons, hover states, and complementary elements.

- **Default**: #7c3aed (Violet 600)
- **Recommendation**: Choose a color that complements your primary color
- **Impact**: Changes the color of secondary buttons, hover states, and complementary elements

### Body Background Color

The background color of the main content area.

- **Default**: #f0f9ff (Sky 50)
- **Recommendation**: Use a light color for better readability
- **Impact**: Changes the background color of the main content area

### Text Color

The default color for body text.

- **Default**: #1f2937 (Gray 800)
- **Recommendation**: Use a dark color for good contrast with the background
- **Impact**: Changes the color of body text throughout the site

### Dark Mode Colors

Colors used when dark mode is active:

#### Dark Mode Background Color

- **Default**: #111827 (Gray 900)
- **Recommendation**: Use a dark color that's not pure black
- **Impact**: Changes the background color in dark mode

#### Dark Mode Text Color

- **Default**: #f3f4f6 (Gray 100)
- **Recommendation**: Use a light color for good contrast with the dark background
- **Impact**: Changes the text color in dark mode

## Typography

The Typography section allows you to customize the fonts used throughout your site:

### Body Font

The font used for body text.

- **Default**: Montserrat, sans-serif
- **Options**: Various Google Fonts
- **Recommendation**: Choose a readable sans-serif font for body text
- **Impact**: Changes the font for all body text

### Heading Font

The font used for headings.

- **Default**: Playfair Display, serif
- **Options**: Various Google Fonts
- **Recommendation**: Choose a distinctive font that complements your body font
- **Impact**: Changes the font for all headings (h1-h6)

### Base Font Size

The base size for body text.

- **Default**: 16px
- **Range**: 12px to 24px
- **Recommendation**: 16px is standard for good readability
- **Impact**: Changes the base font size, which affects all relative font sizes

### Line Height

The default line height for text.

- **Default**: 1.6
- **Range**: 1.0 to 2.0
- **Recommendation**: 1.5 to 1.7 for optimal readability
- **Impact**: Changes the spacing between lines of text

### Font Weight

The weight (boldness) of body text.

- **Default**: 400 (Regular)
- **Options**: 300 (Light), 400 (Regular), 500 (Medium), 600 (Semi-Bold), 700 (Bold)
- **Recommendation**: 400 for body text
- **Impact**: Changes the weight of body text

### Heading Font Weight

The weight of heading text.

- **Default**: 600 (Semi-Bold)
- **Options**: 400 (Regular), 500 (Medium), 600 (Semi-Bold), 700 (Bold)
- **Recommendation**: 600 or 700 for headings
- **Impact**: Changes the weight of heading text

## Advanced Customization

For more advanced customization beyond what's available in the Customizer, you have several options:

### Child Theme

Creating a child theme is the recommended way to make advanced customizations:

1. Create a new directory: `aqualuxe-child`
2. Create a `style.css` file with the following header:
   ```css
   /*
   Theme Name: AquaLuxe Child
   Theme URI: https://example.com/aqualuxe-child/
   Description: Child theme for AquaLuxe
   Author: Your Name
   Author URI: https://example.com/
   Template: aqualuxe-theme
   Version: 1.0.0
   Text Domain: aqualuxe-child
   */
   ```
3. Create a `functions.php` file to add custom functionality

### Custom CSS

For simpler CSS customizations, you can use the Additional CSS section in the Customizer:

1. Go to Appearance > Customize
2. Navigate to Additional CSS
3. Add your custom CSS code
4. See the changes in real-time
5. Click "Publish" to save your changes

### Hooks and Filters

AquaLuxe provides various hooks and filters that you can use to modify the theme's functionality:

```php
// Example: Add content before the header
add_action('aqualuxe_before_header', 'my_custom_header_content');
function my_custom_header_content() {
    echo '<div class="announcement-bar">Special offer: Free shipping on orders over $50!</div>';
}

// Example: Modify excerpt length
add_filter('aqualuxe_excerpt_length', 'my_custom_excerpt_length');
function my_custom_excerpt_length($length) {
    return 30; // 30 words
}
```

For a complete list of available hooks and filters, please refer to the Developer Documentation.

### Template Overrides

You can override any of the theme's templates by creating a copy in your child theme:

1. Locate the template file in the parent theme
2. Create a file with the same name in your child theme
3. Modify the file as needed

For WooCommerce templates, create a `woocommerce` directory in your child theme and place the modified templates there.
