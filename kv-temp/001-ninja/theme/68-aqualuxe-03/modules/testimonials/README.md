# AquaLuxe Testimonials Module

This module provides a custom post type (CPT) for Reviews/Testimonials, with:
- Front-end templates for archive and single testimonials
- Carousel/slider support (JS/CSS stub)
- Submission form and verification system (to be implemented)

## Features
- `aqualuxe_testimonial` CPT
- Front-end templates: `archive-testimonials.php`, `single-testimonial.php`
- JS/CSS for testimonials list and carousel

## Usage
- Add/edit Testimonials in WP Admin
- Front-end displays testimonials list (archive) and single view

## Developer Notes
