# AquaLuxe Auctions/Trade-ins Module

This module provides a custom post type (CPT) for Auctions/Trade-ins, with:
- Front-end templates for archive and single auction items
- JS/CSS for auction timer, bidding, and trade-in (stubs)
- Basic bid form and timer markup

## Usage
- Add/edit Auction items in WP Admin
- Front-end displays auction list (archive) and single view with timer and bid form
