# AquaLuxe Wholesale/B2B Module

This module provides basic wholesale/B2B features for WooCommerce:
- Wholesale customer role
- Wholesale price field for products
- Restrict wholesale-only products to wholesale customers
- (Stub) Registration fields for company/VAT

## Usage
- Assign users to the Wholesale Customer role
- Set wholesale prices in product edit screen
- Wholesale customers see wholesale prices
- Non-wholesale users do not see wholesale-only products
