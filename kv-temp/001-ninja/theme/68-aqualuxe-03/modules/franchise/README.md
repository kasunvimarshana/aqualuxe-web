# AquaLuxe Franchise/Licensing Module

This module provides a custom post type (CPT) for Franchise Locations, with:
- Front-end templates for archive and single franchise locations
- (Stub) Location finder, map, application form, territory management

## Usage
- Add/edit Franchise Locations in WP Admin
- Front-end displays franchise list (archive) and single view
