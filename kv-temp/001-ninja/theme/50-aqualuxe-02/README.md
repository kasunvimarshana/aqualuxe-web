# AquaLuxe WordPress Theme

AquaLuxe is a premium WordPress theme designed specifically for aquatic businesses such as aquarium shops, pool supply stores, and marine equipment retailers. The theme offers a modern, responsive design with extensive WooCommerce integration and a range of advanced features.

## Features

### Core Features

- **Responsive Design**: Fully responsive layout that looks great on all devices
- **Dark Mode**: Built-in dark mode toggle with user preference persistence
- **Multilingual Support**: Compatible with WPML and Polylang for multilingual sites
- **Multi-Currency Support**: Support for multiple currencies in WooCommerce
- **Customizer Options**: Extensive theme customization options through the WordPress Customizer
- **Modular Architecture**: Clean, modular code structure for easy customization and extension

### WooCommerce Integration

- **Enhanced Product Pages**: Custom-designed product pages with additional features
- **Quick View**: Product quick view modal for faster shopping
- **Optimized Cart & Checkout**: Streamlined cart and checkout process for better conversion
- **Account Dashboard**: Comprehensive customer account dashboard
- **Order Tracking**: Advanced order tracking functionality with shipment tracking integration
- **Wishlist**: Built-in wishlist functionality
- **Product Comparison**: Feature-rich product comparison tool

### Page Templates

- **Homepage Template**: Flexible homepage with multiple customizable sections
- **About Page Template**: Custom template for about pages
- **Services Page Template**: Template for showcasing services
- **Contact Page Template**: Contact page with Google Maps integration and contact form
- **FAQ Page Template**: FAQ page with accordion sections and category filtering
- **Legal Pages Templates**: Templates for privacy policy, terms & conditions, and return policy

### Advanced Features

- **Newsletter Integration**: Built-in newsletter subscription functionality
- **Social Media Integration**: Social media profile links and sharing buttons
- **Demo Content Importer**: One-click demo content import
- **SEO Optimization**: Schema.org structured data and Open Graph tags
- **Accessibility Features**: WCAG compliance features for better accessibility
- **Performance Optimization**: Optimized for speed and performance
- **Security Best Practices**: Enhanced security features

## Requirements

- WordPress 5.6 or higher
- PHP 7.4 or higher
- MySQL 5.6 or higher
- WooCommerce 5.0 or higher (for e-commerce functionality)

## Installation

1. Upload the `aqualuxe-theme` folder to the `/wp-content/themes/` directory
2. Activate the theme through the 'Themes' menu in WordPress
3. Install required plugins when prompted
4. Import demo content (optional) through Appearance > Demo Import
5. Configure theme options through Appearance > Customize
