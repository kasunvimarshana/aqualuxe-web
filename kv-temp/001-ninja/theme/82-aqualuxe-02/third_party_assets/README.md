# Third-party Assets (CC0/Permissive Only)

Use this folder to collect links and notes for assets you plan to include (audio, images, 3D models, fonts). Keep raw downloads in `assets/src/<type>` under the theme so they are included in the build, and document the source and license here.

- Only use CC0, Public Domain, or permissive licenses (e.g., MIT). Avoid attribution-required assets unless you manage attribution clearly in the site footer.
- Verify licenses on the source page; double-check there are no usage restrictions.

Recommended sources:
- OpenGameArt: https://opengameart.org/ (filter by CC0)
- Kenney: https://kenney.nl/assets (CC0 / public domain)
- Wikimedia Commons: https://commons.wikimedia.org/ (check license per file)
- Freesound: https://freesound.org/ (filter by CC0)

Place the files here for review, then copy into:
- Audio: `assets/src/audio/`
- Images: `assets/src/images/`
- Models: `assets/src/models/`
- Fonts: `assets/src/fonts/`

After copying, rebuild assets to populate `assets/dist/`.
