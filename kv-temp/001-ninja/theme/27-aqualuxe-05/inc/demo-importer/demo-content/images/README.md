# Demo Images Directory

This directory contains all images used by the demo content importer for the AquaLuxe theme. These images are properly licensed for use in demo content.

## Image Categories

The following image categories are included:

- **Betta Fish**: Images of betta fish for care guides and product showcases
- **Discus Fish**: Images of discus fish for care guides and product showcases
- **Neon Tetra**: Images of neon tetra fish for care guides and product showcases
- **Planted Aquarium**: Images of planted aquariums for project showcases
- **Products**: Images of aquarium products for the WooCommerce store
- **Team**: Images of team members for the team section
- **Projects**: Images of completed aquarium projects for the portfolio section

## Image Naming Conventions

- All images use descriptive filenames that match the references in the JSON data files
- Filenames use lowercase letters, numbers, and hyphens (no spaces)
- Images include the type in the filename (e.g., "betta-fish-1.jpg", "discus-fish-2.jpg")

## Image Sources and Attribution

All images are sourced from Pexels and are free to use under the Pexels license. See the ATTRIBUTION.md file in this directory for detailed attribution information.

## Important Notes

- All demo images are for demonstration purposes only
- Users should replace demo images with their own content before publishing their site
- Each image includes proper attribution metadata when imported
- Maximum file size: 500KB per image
- Image dimensions:
  - Featured images: 1200x800px
  - Product images: 1000x1000px
  - Gallery images: 800x600px
