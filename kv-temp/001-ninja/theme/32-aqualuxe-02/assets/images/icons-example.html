icon-star

How to Use

Include the SVG sprite in your HTML:

<!-- Include at the beginning of your body tag --> <svg xmlns="http://www.w3.org/2000/svg" style="display: none;"> <!-- You can include the sprite inline or via PHP include --> <?php include get_template_directory() . '/assets/images/sprite.svg'; ?> </svg>

Use an icon in your HTML:

<svg class="icon" aria-hidden="true"> <use xlink:href="#icon-name"></use> </svg>

Styling icons with CSS:

.icon { width: 24px; height: 24px; fill: currentColor; /* Inherits color from parent */ } /* You can also style specific icons */ .icon-search { fill: #0077b6; }
