# AquaLuxe Demo Importer Module

This module provides an admin interface to import demo content, widgets, and theme options for the AquaLuxe theme.

## Features
- Admin submenu under Appearance > Demo Importer
- One-click import for demo content (WXR/XML), widgets, and theme options
- Extensible for multiple demo presets

## Usage
1. Go to Appearance > Demo Importer in the WordPress admin.
2. Click "Import Demo Content" to import the default demo data.
