# AquaLuxe Testimonials Module

This module provides a custom post type for testimonials, a star rating system, frontend submission form, and a Swiper.js carousel for display.

## Features
- Custom post type: Testimonial
- Star rating meta box
- Frontend testimonial submission form (shortcode)
- Carousel/slider display (Swiper.js)
- Pending status for admin verification

## Usage
- Use shortcode [aqualuxe_testimonial_form] to display the submission form.
- Display testimonials with a custom template or block.
