# Installation Guide

This guide will walk you through the process of installing the AquaLuxe theme.

## Requirements

*   WordPress 5.0 or higher
*   PHP 7.4 or higher
*   WooCommerce 5.0 or higher (optional, but recommended for e-commerce features)

## Installation Steps

1.  **Download the Theme:** Download the latest version of the AquaLuxe theme as a ZIP file.
2.  **Upload to WordPress:**
    *   In your WordPress admin, go to `Appearance > Themes`.
    *   Click `Add New`, then `Upload Theme`.
    *   Choose the `aqualuxe.zip` file you downloaded and click `Install Now`.
3.  **Activate the Theme:** Once the theme is installed, click `Activate`.

## Required Plugins

For full functionality, the following plugins are recommended:

*   **WooCommerce:** For e-commerce capabilities.
*   **Polylang:** For multilingual support.
*   **WooCommerce Subscriptions:** For recurring payments.
*   **WooCommerce Bookings:** For booking services.
*   **The Events Calendar:** For managing events.
*   **Contact Form 7:** For the contact form.

Install these plugins from the WordPress plugin repository by going to `Plugins > Add New`.

## Demo Content Import

After activating the theme and required plugins, you can import the demo content to get a starting point for your site.

1.  Go to `Appearance > Demo Importer`.
2.  Click the `Import Demo Content` button.
3.  Wait for the import process to complete. This may take a few minutes.

Your site should now be set up with the demo content, including pages, posts, products, and images.
