# AquaLuxe WordPress Theme

![AquaLuxe Theme](screenshot.png)

## Description

AquaLuxe is a premium WordPress theme designed specifically for luxury aquatic retail businesses. With its elegant design and comprehensive feature set, AquaLuxe provides everything you need to create a stunning online presence for your aquatic business.

**Tagline:** Bringing elegance to aquatic life – globally

## Features

### Core Features

- **Multitenant Architecture**: Support for WordPress multisite installations
- **Multivendor Support**: Compatible with popular marketplace plugins
- **Multilingual Support**: Built-in support for multiple languages
- **Multi-currency Support**: Sell products in different currencies
- **Dark Mode**: Elegant dark theme with persistent user preferences
- **Responsive Design**: Looks great on all devices, from mobile to desktop
- **Modular Design**: Easily customize and extend the theme

### WooCommerce Integration

- **Dual-state Architecture**: Works with or without WooCommerce enabled
- **Custom Product Tabs**: Care Guide and Shipping tabs for aquatic products
- **Quick View**: View product details without leaving the page
- **Wishlist**: Allow customers to save products for later
- **AJAX Add to Cart**: Add products to cart without page reload
- **Sticky Add to Cart**: Always visible add to cart button on product pages
- **Advanced Filtering**: Filter products by various attributes

### Performance & SEO

- **Optimized Code**: Clean, well-structured code for optimal performance
- **SEO Friendly**: Built with search engine optimization in mind
- **Fast Loading**: Minimal dependencies and optimized assets
- **Accessibility**: WCAG compliant for better accessibility

## Requirements

- WordPress 5.9 or higher
- PHP 7.4 or higher
- MySQL 5.6 or higher
- WooCommerce 6.0 or higher (optional)

## Installation

1. Upload the `aqualuxe-theme` folder to the `/wp-content/themes/` directory
2. Activate the theme through the 'Themes' menu in WordPress
3. Configure theme settings through the WordPress Customizer
