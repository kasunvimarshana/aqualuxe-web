# Installation Guide

This guide will walk you through the process of installing the AquaLuxe Pro theme.

## Prerequisites

*   A working WordPress installation.
*   Access to the WordPress admin panel.

## Installation Steps

1.  **Download the Theme:**
    *   Download the latest version of the AquaLuxe Pro theme as a zip file from the official repository.

2.  **Upload and Install:**
    *   Log in to your WordPress admin panel.
    *   Navigate to **Appearance > Themes**.
    *   Click the **Add New** button at the top of the page.
    *   Click the **Upload Theme** button.
    *   Click **Choose File** and select the `aqualuxe-pro.zip` file you downloaded.
    *   Click **Install Now**.

3.  **Activate the Theme:**
    *   After the installation is complete, click the **Activate** link.

## Post-Installation Steps

*   **Install Recommended Plugins:**
    *   After activating the theme, you will see a notice to install recommended plugins, including WooCommerce. Follow the prompts to install and activate them.

*   **Import Demo Content:**
    *   To get started quickly, you can import the demo content.
    *   Navigate to **Appearance > Import Demo Data**.
    *   Click the **Import Demo Data** button and wait for the process to complete.

*   **Configure Theme Options:**
    *   Go to **Appearance > Customize** to configure the theme's settings, such as your logo, colors, and fonts.

Congratulations! You have successfully installed and set up the AquaLuxe Pro theme.
