## Introduction

Welcome to AquaLuxe, a modern and feature-rich WordPress theme designed for businesses, online stores, and blogs. This user guide will help you set up and customize your website using the AquaLuxe theme.

### Key Features

- **Modern Design**: Clean, professional design with attention to detail
- **WooCommerce Ready**: Fully compatible with WooCommerce for online stores
- **Responsive Layout**: Looks great on all devices, from mobile to desktop
- **Dark Mode**: Built-in dark mode toggle for better user experience
- **Multilingual Support**: Ready for translation into any language
- **SEO Optimized**: Built with best SEO practices in mind
- **Performance Focused**: Fast loading times and optimized code
- **Accessibility Compliant**: Follows WCAG guidelines for accessibility

## Installation

### Requirements

- WordPress 5.8 or higher
- PHP 7.4 or higher
- MySQL 5.6 or higher

### Installation Steps

1. **Upload the Theme**
   - Log in to your WordPress admin panel
   - Navigate to Appearance > Themes
   - Click "Add New"
   - Click "Upload Theme"
   - Choose the aqualuxe.zip file
   - Click "Install Now"

2. **Activate the Theme**
   - After installation, click "Activate"
   - You'll be redirected to the AquaLuxe welcome page

3. **Install Required Plugins**
   - After activation, you'll see a notice at the top of your admin panel
   - Click "Begin installing plugins"
   - Select all recommended plugins
   - Click "Install"
   - After installation, activate all plugins

3. **Configure Typography**

   - Click "Publish" to save changes

4. **Configure Layout**
   - Navigate to Appearance > Customize
   - Click on "Layout"
   - Set container width and spacing
   - Configure sidebar position
   - Click "Publish" to save changes

## Customizing Your Site

### Using the Customizer

The WordPress Customizer is the main tool for customizing your AquaLuxe theme:

1. Navigate to Appearance > Customize
2. Use the left sidebar to navigate between different customization sections
3. Preview changes in real-time on the right
4. Click "Publish" to save your changes

### Customizer Sections

AquaLuxe adds the following sections to the Customizer:

- **Site Identity**: Logo, site title, tagline, and favicon
- **Colors**: Primary, secondary, accent, text, and background colors
- **Typography**: Heading and body fonts, font sizes
- **Layout**: Container width, sidebar position, spacing
- **Header**: Header layout, sticky header, transparent header
- **Footer**: Footer layout, widgets, copyright text
- **Blog**: Blog layout, post meta, featured images
- **WooCommerce**: Shop layout, product cards, checkout
- **Advanced**: Custom CSS, performance settings

### Global Settings

These settings affect your entire website:

- **Container Width**: Controls the maximum width of your content
- **Button Style**: Choose between different button styles
- **Card Style**: Choose between different card styles
- **Border Radius**: Set the roundness of elements like buttons and cards
- **Box Shadows**: Enable or disable shadows on cards and other elements

## Header & Navigation

AquaLuxe includes multiple header styles:

1. **Default Header**: Logo on the left, navigation on the right

To enable or disable these features:

1. Navigate to Appearance > Customize
2. Click on "Header"
3. Toggle the features you want to enable or disable
4. Click "Publish" to save changes

## Homepage Setup

### Using the Homepage Template

AquaLuxe includes a dedicated homepage template:

1. Create a new page (Pages > Add New)
2. Give it a title (e.g., "Home")
3. In the Page Attributes panel, select "Homepage" as the template
4. Publish the page
5. Navigate to Settings > Reading
6. Set "Your homepage displays" to "A static page"
7. Select your new page as the Homepage
8. Click "Save Changes"

### Using the Block Editor

The homepage template works best with the block editor:

1. Edit your homepage
2. Add blocks for different sections:
   - Hero section (Cover block)
   - Features section (Columns block)
   - About section (Media & Text block)
   - Services section (Columns or Gallery block)
   - Testimonials section (Quote blocks in Columns)
   - Call to action section (Cover block with Button)
3. Use the AquaLuxe blocks for specialized sections

### Using a Page Builder

AquaLuxe is compatible with popular page builders:

- **Elementor**: Use the Elementor templates included with AquaLuxe
- **Beaver Builder**: Use the Beaver Builder templates included with AquaLuxe
- **Gutenberg**: Use the block patterns included with AquaLuxe

## Dark Mode

### Enabling Dark Mode

AquaLuxe includes a built-in dark mode feature:

1. Navigate to Appearance > Customize
2. Click on "Advanced"
3. Under "Dark Mode", toggle "Enable Dark Mode"

The dark mode toggle appears in the header by default. To change its position:

1. Navigate to Appearance > Customize
2. Click on "Advanced"
3. Under "Dark Mode", select the toggle position:

   - Footer
   - Floating
   - None (disable toggle)
4. Click "Publish" to save changes

To customize the dark mode colors:

1. Navigate to Appearance > Customize

## Troubleshooting

### Common Issues

#### Theme not displaying correctly

If the theme is not displaying correctly:

1. Clear your browser cache
2. Disable conflicting plugins
3. Check for JavaScript errors in the browser console
4. Make sure you're using the latest version of WordPress
5. Make sure you're using the latest version of AquaLuxe

#### WooCommerce issues

If you're experiencing WooCommerce issues:

1. Make sure WooCommerce is up to date
2. Reset WooCommerce settings to default
3. Clear the product cache
4. Disable conflicting plugins
5. Check for JavaScript errors in the browser console

#### Customizer not saving changes

If the Customizer is not saving changes:

1. Clear your browser cache
2. Try a different browser
3. Temporarily disable caching plugins
4. Check for JavaScript errors in the browser console
5. Make sure you have sufficient permissions
