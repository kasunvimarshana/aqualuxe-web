## Introduction

AquaLuxe offers extensive customization options through the WordPress Customizer, allowing you to tailor your website's appearance and functionality without writing code. This document provides a comprehensive guide to all available customization options.

## Accessing the Customizer

To access the WordPress Customizer:

1. Log in to your WordPress admin dashboard
2. Go to **Appearance > Customize**
3. The Customizer will open with a live preview of your site
4. Make changes using the left sidebar controls
5. Click **Publish** to save your changes

## Site Identity

The Site Identity section allows you to set up your site's branding elements.

### Logo

- **Upload Logo**: Upload your site logo (recommended size: 250×100 px)
- **Logo Width**: Adjust the display width of your logo
- **Logo Height**: Adjust the display height of your logo
- **Mobile Logo**: Upload a separate logo for mobile devices (optional)
- **Retina Logo**: Upload a 2x size logo for high-resolution displays (optional)

### Site Title & Tagline

- **Site Title**: Your website's name
- **Tagline**: A short description of your site
- **Display Site Title**: Toggle to show/hide the site title
- **Display Tagline**: Toggle to show/hide the tagline

### Site Icon

- **Site Icon (Favicon)**: Upload a small icon (at least 512×512 px) that represents your site

## Colors

The Colors section allows you to customize your site's color scheme.

### Color Schemes

- **Predefined Schemes**: Choose from several predefined color schemes
- **Custom Scheme**: Create your own color scheme

### Primary Colors

- **Primary Color**: Main brand color used throughout the site
- **Secondary Color**: Supporting color used for accents and highlights
- **Tertiary Color**: Additional color for variety and contrast

### Text Colors

- **Body Text Color**: Default text color for paragraphs and general content
- **Heading Color**: Color for headings (h1-h6)
- **Link Color**: Color for hyperlinks
- **Link Hover Color**: Color for hyperlinks when hovered

### Component Colors

- **Button Background**: Background color for buttons
- **Button Text**: Text color for buttons
- **Button Hover Background**: Background color for buttons when hovered
- **Button Hover Text**: Text color for buttons when hovered

### Dark Mode Colors

- **Dark Mode Background**: Background color when dark mode is active
- **Dark Mode Text**: Text color when dark mode is active
- **Dark Mode Accent**: Accent color when dark mode is active

## Typography

The Typography section allows you to customize fonts and text settings.

### Font Families

- **Body Font**: Font for general text content
  - System fonts (Arial, Helvetica, Georgia, etc.)
  - Google Fonts (400+ options)
  - Custom fonts (via Custom CSS)
- **Heading Font**: Font for headings (h1-h6)
  - Same options as Body Font
- **Menu Font**: Font for navigation menus
  - Same options as Body Font

### Font Sizes

- **Base Font Size**: Base size for body text (default: 16px)
- **Heading Sizes**: Individual sizes for h1-h6 elements
- **Menu Font Size**: Size for menu items
- **Button Font Size**: Size for button text

### Font Weights

- **Body Font Weight**: Weight for general text (normal, bold, etc.)
- **Heading Font Weight**: Weight for headings
- **Menu Font Weight**: Weight for menu items
- **Button Font Weight**: Weight for button text

### Line Heights

- **Body Line Height**: Line spacing for paragraphs (default: 1.6)
- **Heading Line Height**: Line spacing for headings
- **Menu Line Height**: Line spacing for menu items

## Header Settings

The Header Settings section allows you to customize your site's header.

### Header Layout

- **Header Style**: Choose from multiple header layouts
  - **Standard**: Logo on left, menu on right
  - **Centered**: Logo centered, menu below
  - **Split**: Logo centered, menu split on both sides
  - **Transparent**: Overlay header on page content

### Header Options

- **Sticky Header**: Enable/disable sticky header on scroll
- **Header Height**: Adjust header height (desktop and mobile)
- **Header Padding**: Adjust padding around header elements
- **Header Background**: Set header background color or image
- **Header Border**: Add optional border to header

### Top Bar

- **Enable Top Bar**: Show/hide the top bar above the main header
- **Top Bar Content**: Add custom content to the top bar
  - Contact information
  - Secondary menu
  - Social icons
  - Custom text/HTML
- **Top Bar Background**: Set top bar background color
- **Top Bar Text Color**: Set top bar text color

### Navigation

- **Menu Position**: Set menu position within header
- **Menu Alignment**: Align menu items (left, center, right)
- **Dropdown Style**: Choose dropdown menu style
- **Mobile Menu Style**: Choose mobile menu style
  - Off-canvas
  - Dropdown
  - Full screen

## Footer Settings

The Footer Settings section allows you to customize your site's footer.

### Footer Layout

- **Footer Style**: Choose from multiple footer layouts
  - **Standard**: Widget areas with copyright below
  - **Centered**: Centered content with social icons
  - **Complex**: Multiple widget areas with newsletter signup

### Footer Widgets

- **Widget Columns**: Choose number of widget columns (1-6)
- **Widget Areas**: Configure individual widget areas
- **Widget Alignment**: Set alignment for widget content

### Footer Options

- **Footer Background**: Set footer background color or image
- **Footer Text Color**: Set footer text color
- **Footer Padding**: Adjust padding around footer elements
- **Footer Border**: Add optional border to footer

### Copyright Bar

- **Copyright Text**: Set custom copyright text (supports HTML)
- **Copyright Bar Background**: Set copyright bar background color
- **Copyright Bar Text Color**: Set copyright bar text color
- **Payment Icons**: Add payment method icons (for e-commerce sites)

## Advanced Customization

The Advanced Customization section allows for more technical customizations.

### Custom CSS

- **Additional CSS**: Add custom CSS rules
- **Custom CSS Classes**: Define reusable CSS classes

### Custom JavaScript

- **Header Scripts**: Add custom JavaScript to header
- **Footer Scripts**: Add custom JavaScript to footer

### Code Injection

- **Header Code**: Add custom code to the `<head>` section
- **Body Open Code**: Add custom code after the `<body>` tag
- **Footer Code**: Add custom code before the closing `</body>` tag

## Customizer Export/Import

AquaLuxe includes functionality to export and import your customizer settings.

### Exporting Settings

1. Go to **Appearance > Customize > Export/Import**
2. Click **Export Settings**
3. Save the downloaded JSON file

### Importing Settings

1. Go to **Appearance > Customize > Export/Import**
2. Click **Choose File** and select your JSON file
3. Click **Import Settings**
4. Review and confirm the changes

### Preset Configurations

AquaLuxe includes several preset configurations for different types of websites:

- **Business**: Professional layout for business websites
- **E-commerce**: Optimized for online stores
- **Blog**: Focused on content and readability
- **Portfolio**: Showcase creative work
- **Corporate**: Formal design for corporate sites

To apply a preset:
1. Go to **Appearance > Customize > Export/Import**
2. Select a preset from the dropdown
3. Click **Apply Preset**
4. Review and confirm the changes
