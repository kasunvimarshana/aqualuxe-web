# AquaLuxe Theme

A premium WordPress theme for ornamental aquatic solutions.

## Installation

1. Upload the `aqualuxe` theme folder to `wp-content/themes/`.
2. Activate the theme in the WordPress admin panel.
3. Install required plugins.
4. Import demo content (optional).

## Development

1. Run `npm install` to install dependencies.
2. Run `npm run watch` to start development.

## Build

Run `npm run prod` to build for production.
