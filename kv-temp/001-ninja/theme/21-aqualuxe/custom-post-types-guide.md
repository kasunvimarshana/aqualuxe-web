## Understanding Custom Post Types

Custom Post Types (CPTs) are specialized content types in WordPress beyond the standard posts and pages. They allow you to organize and present specific types of content in ways that make sense for your business.

Benefits of using custom post types:
- Organize content by type
- Apply specific features and fields to different content types
- Create specialized archives and single post layouts
- Separate content management in the WordPress admin

## Services Post Type

The Services post type allows you to showcase the various aquarium services you offer, such as tank setup, maintenance, aquascaping, or consultation.

### Creating a Service

1. In your WordPress admin, go to **Services > Add New**
2. Enter a title for your service (e.g., "Custom Aquarium Design")
3. In the main content editor, add a detailed description of the service
4. In the Service Details meta box, enter:
   - **Service Icon**: Select an icon that represents the service
   - **Price**: Enter the price or price range (e.g., "$99" or "From $199")
   - **Duration**: Enter the typical duration (e.g., "2 hours" or "3-4 weeks")
   - **Service Color**: Select a color for the service card
5. Add a featured image that represents the service
6. In the right sidebar:
   - **Service Categories**: Assign relevant categories
   - **Featured Service**: Check this to highlight important services
7. Click "Publish" to save your service

### Service Archive Page

The Services archive page displays all your services in a grid or list format. To customize this page:

1. Go to **Appearance > Customize > Services Archive**
2. Configure options such as:
   - Layout style (Grid or List)
   - Number of columns
   - Items per page
   - Sorting options
   - Display of service details

### Single Service Page

Each service has its own detailed page. The default layout includes:
- Service title and featured image
- Service details (icon, price, duration)
- Main content description
- Related services
- Call-to-action section

To customize the single service template:
1. Go to **Appearance > Customize > Single Service**
2. Configure options for layout, sidebar, and display elements

## Projects Post Type

The Projects post type allows you to showcase your completed aquarium installations, designs, and other projects.

### Creating a Project

1. In your WordPress admin, go to **Projects > Add New**
2. Enter a title for your project (e.g., "Corporate Office Reef Tank")
3. In the main content editor, add a detailed description of the project
4. In the Project Details meta box, enter:
   - **Client**: Enter the client name or company
   - **Completion Date**: When the project was completed
   - **Project Type**: Type of installation (e.g., "Reef Tank," "Planted Aquarium")
   - **Project Size**: Size or volume of the aquarium
   - **Project Budget Range**: Optional budget range information
5. In the Project Gallery meta box:
   - Click "Add Images" to upload multiple project photos
   - Arrange images in the desired order
6. Add a featured image (this will be the main project thumbnail)
7. In the right sidebar:
   - **Project Categories**: Assign relevant categories
   - **Featured Project**: Check this to highlight important projects
8. Click "Publish" to save your project

### Project Archive Page

The Projects archive page displays all your projects in a portfolio grid. To customize this page:

1. Go to **Appearance > Customize > Projects Archive**
2. Configure options such as:
   - Layout style (Grid, Masonry, or Carousel)
   - Number of columns
   - Items per page
   - Filtering options
   - Animation effects

### Single Project Page

Each project has its own detailed page. The default layout includes:
- Project title and featured image
- Project details (client, date, type, etc.)
- Image gallery
- Main content description
- Related projects
- Call-to-action section

To customize the single project template:
1. Go to **Appearance > Customize > Single Project**
2. Configure options for layout, gallery style, and display elements

## Team Members Post Type

The Team Members post type allows you to showcase your staff, experts, and specialists.

### Creating a Team Member

1. In your WordPress admin, go to **Team > Add New**
2. Enter the team member's name as the title
3. In the main content editor, add a detailed bio
4. In the Team Member Details meta box, enter:
   - **Position**: Job title or role
   - **Email**: Contact email address
   - **Phone**: Contact phone number
   - **Expertise**: Areas of specialization (comma-separated)
   - **Social Media**: Links to social profiles
5. Add a professional photo as the featured image
6. In the right sidebar:
   - **Team Categories**: Assign relevant departments or categories
   - **Featured Member**: Check this to highlight key team members
   - **Team Order**: Set a number to control display order
7. Click "Publish" to save the team member

### Team Archive Page

The Team archive page displays all your team members in a grid. To customize this page:

1. Go to **Appearance > Customize > Team Archive**
2. Configure options such as:
   - Layout style (Grid or List)
   - Number of columns
   - Items per page
   - Sorting options
   - Display of member details

### Single Team Member Page

Each team member has their own detailed page. The default layout includes:
- Name, position, and photo
- Contact information and social media links
- Detailed biography
- Areas of expertise
- Related team members

To customize the single team member template:
1. Go to **Appearance > Customize > Single Team Member**
2. Configure options for layout and display elements

## Events Post Type

The Events post type allows you to promote workshops, seminars, exhibitions, and other events related to your aquarium business.

### Creating an Event

1. In your WordPress admin, go to **Events > Add New**
2. Enter the event title
3. In the main content editor, add a detailed description of the event
4. In the Event Details meta box, enter:
   - **Event Date**: Select the date of the event
   - **Event Time**: Enter the time (e.g., "6:00 PM - 9:00 PM")
   - **Event Location**: Enter the venue address or virtual event details
   - **Event Cost**: Enter the price or "Free"
   - **Registration Required**: Select Yes or No
   - **Registration URL**: If applicable, add the link to register
5. Add an event image as the featured image
6. In the right sidebar:
   - **Event Categories**: Assign relevant categories
   - **Featured Event**: Check this to highlight important events
7. Click "Publish" to save your event

### Event Archive Page

The Events archive page displays all your upcoming and past events. To customize this page:

1. Go to **Appearance > Customize > Events Archive**
2. Configure options such as:
   - Layout style (List, Grid, or Calendar)
   - Sorting options (upcoming first or chronological)
   - Display of event details
   - Past events display

### Single Event Page

Each event has its own detailed page. The default layout includes:
- Event title and featured image
- Event details (date, time, location, cost)
- Registration button (if applicable)
- Main content description
- Event countdown timer
- Related events
- Google Maps integration (for physical locations)

To customize the single event template:
1. Go to **Appearance > Customize > Single Event**
2. Configure options for layout and display elements

## Testimonials Post Type

The Testimonials post type allows you to showcase customer reviews and feedback.

### Creating a Testimonial

1. In your WordPress admin, go to **Testimonials > Add New**
2. Enter a title for the testimonial (can be a summary or the client's name)
3. In the main content editor, add the full testimonial text
4. In the Testimonial Details meta box, enter:
   - **Client Name**: Name of the person giving the testimonial
   - **Client Location**: City, company, or other identifier
   - **Client Rating**: Select a rating from 1-5 stars
   - **Project Type**: What service the testimonial relates to
5. Optionally add a client photo as the featured image
6. In the right sidebar:
   - **Featured Testimonial**: Check this to highlight important testimonials
   - **Project Link**: Optionally link to a related project
7. Click "Publish" to save your testimonial

### Testimonial Archive Page

The Testimonials archive page displays all your testimonials. To customize this page:

1. Go to **Appearance > Customize > Testimonials Archive**
2. Configure options such as:
   - Layout style (Grid, List, or Carousel)
   - Number of columns
   - Items per page
   - Display of testimonial details

### Displaying Testimonials

Unlike other post types, testimonials are typically displayed as elements on other pages rather than having their own detailed single pages. You can:

1. Use the Testimonials widget in sidebar or footer areas
2. Add the Testimonials section to your homepage
3. Use the `[aqualuxe_testimonials]` shortcode in any page or post
4. Add testimonials to service or project pages

## Displaying Custom Post Types

AquaLuxe provides multiple ways to display your custom post types throughout your site:

### Archive Pages

Each custom post type has its own archive page that displays all entries:
- Services: `yourdomain.com/services/`
- Projects: `yourdomain.com/projects/`
- Team: `yourdomain.com/team/`
- Events: `yourdomain.com/events/`
- Testimonials: `yourdomain.com/testimonials/`

### Shortcodes

Use these shortcodes to display custom post types anywhere:

```
[aqualuxe_services count="3" columns="3" style="grid" category="maintenance"]
[aqualuxe_projects count="6" columns="3" style="masonry" category="reef-tanks"]
[aqualuxe_team count="4" columns="4" category="specialists"]
[aqualuxe_events count="3" style="list" upcoming="true"]
[aqualuxe_testimonials count="3" style="slider" rating="5"]
```

### Widgets

AquaLuxe includes widgets for each custom post type:
1. Go to **Appearance > Widgets**
2. Find widgets like "AquaLuxe: Services," "AquaLuxe: Projects," etc.
3. Add them to any widget area and configure the options

### Page Templates

AquaLuxe includes specialized page templates for custom post types:
1. Create a new page
2. In the Page Attributes section, select a template:
   - Services Template
   - Projects Template
   - Team Template
   - Events Template
3. Publish the page to use the specialized layout

### Elementor Integration

If you use Elementor page builder:
1. Edit a page with Elementor
2. Find the AquaLuxe widgets in the elements panel
3. Drag and drop widgets like "AquaLuxe Services," "AquaLuxe Projects," etc.
4. Configure the widget options

## Customizing Post Types

### Adding Custom Fields

To add custom fields to any post type:

1. Install and activate the Advanced Custom Fields (ACF) plugin
2. Go to **Custom Fields > Field Groups > Add New**
3. Create your custom fields
4. In "Location," set rules to show the field group on your desired post type
5. Publish the field group

To display custom fields in your theme:

```php
<?php 
if (function_exists('get_field')) {
    $custom_value = get_field('your_field_name');
    echo $custom_value;
}
?>
```

### Modifying Post Type Features

To modify post type features in a child theme:

1. Create a child theme (if you haven't already)
2. In your child theme's `functions.php`, add code to modify post types:

```php
function aqualuxe_child_modify_post_types() {
    // Modify the Services post type
    add_post_type_support('service', 'excerpt'); // Add excerpt support
    
    // Change the menu icon for Projects
    global $wp_post_types;
    if (isset($wp_post_types['project'])) {
        $wp_post_types['project']->menu_icon = 'dashicons-portfolio';
    }
}
add_action('init', 'aqualuxe_child_modify_post_types', 20);
```

### Customizing Archive and Single Templates

To customize how post types display:

1. Create a child theme (if you haven't already)
2. Create template files in your child theme following WordPress template hierarchy:
   - For archives: `archive-{post_type}.php` (e.g., `archive-service.php`)
   - For single posts: `single-{post_type}.php` (e.g., `single-project.php`)
3. Customize these templates as needed

## Importing Demo Content

AquaLuxe includes demo content for all custom post types to help you get started quickly.

### Using the One-Click Demo Importer

1. Go to **Appearance > AquaLuxe Setup**
2. Click on the "Import Demo Content" tab
3. Click "Import Demo Content" button
4. Wait for the import to complete

### Selective Import

To import only specific post types:

1. Go to **Appearance > AquaLuxe Setup > Import Demo Content**
2. Expand the "Selective Import" section
3. Check only the post types you want to import
4. Click "Import Selected Content"

### Manual Import

If you prefer to manually create content:

1. Refer to the demo content files in `/demo-content/` directory
2. Use these as templates to create your own entries
3. Follow the structure and formatting of the demo content
