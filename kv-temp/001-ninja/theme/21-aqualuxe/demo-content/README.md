# AquaLuxe WordPress Theme Demo Content

This directory contains demo content for the AquaLuxe WordPress theme, designed to showcase the theme's features and provide a starting point for users. The content is organized by post type and can be imported into a WordPress installation using the provided import tools.

## Directory Structure

- `/images` - Contains images used in the demo content
- `/products` - WooCommerce product data in JSON format
- `/posts` - Blog post content in JSON format
- `/services` - Custom post type content for Services
- `/events` - Custom post type content for Events
- `/projects` - Custom post type content for Projects
- `/testimonials` - Custom post type content for Testimonials
- `/team` - Custom post type content for Team Members

## Using the Demo Content

### Option 1: Manual Import
1. Create new posts/pages/products in WordPress
2. Copy and paste the content from the JSON files
3. Upload and attach the corresponding images

### Option 2: Import via WP All Import Plugin
1. Install and activate the WP All Import plugin
2. Create a new import for each post type
3. Select the corresponding JSON file
4. Map the JSON fields to WordPress fields
5. Run the import

### Option 3: Use the AquaLuxe Demo Importer
If you're using the AquaLuxe theme, it includes a one-click demo importer:
1. Go to Appearance > AquaLuxe Setup
2. Click on the "Import Demo Content" tab
3. Click the "Import Demo Content" button
4. Wait for the import to complete

## Customizing the Demo Content
