## Introduction

Welcome to AquaLuxe, a premium WordPress theme designed specifically for ornamental fish farming businesses. This user guide will help you set up and customize your website using the AquaLuxe theme.

### Theme Features
- Modern, responsive design
- WooCommerce integration for online sales
- Dark mode support
- Multilingual capabilities
- Custom post types for fish species, care guides, team members, testimonials, and FAQs
- Advanced product filtering
- Wishlist functionality
- Quick view for products
- SEO optimization with schema.org markup
- Performance optimization with image compression and caching

## Theme Installation

### Requirements
- WordPress 5.8 or higher
- PHP 7.4 or higher
- WooCommerce 6.0 or higher (if you plan to sell products)

### Installation Steps
1. Log in to your WordPress admin dashboard
2. Go to **Appearance > Themes**
3. Click **Add New**
4. Click **Upload Theme**
5. Click **Choose File** and select the `aqualuxe.zip` file
6. Click **Install Now**
7. After installation, click **Activate**

### Plugin Requirements
AquaLuxe works best with the following plugins:
- WooCommerce (for online store functionality)
- Elementor (for drag-and-drop page building)
- Contact Form 7 (for contact forms)
- WPML or Polylang (for multilingual support)
- Yoast SEO or Rank Math (for additional SEO features)

## Theme Setup

After activating the theme, you'll see a notice at the top of your dashboard recommending the installation of required and optional plugins. Follow these steps to complete the setup:

1. Click on **Begin installing plugins**
2. Select all the recommended plugins
3. Click **Install** and then **Activate**
4. After plugin installation, go to **AquaLuxe > Setup Wizard** to configure basic theme settings

### Setup Wizard
The setup wizard will guide you through the following steps:
1. **Welcome**: Introduction to the setup process
2. **Plugins**: Install and activate required plugins
3. **Demo Import**: Option to import demo content
4. **Theme Options**: Basic theme settings
5. **WooCommerce**: Basic store setup (if WooCommerce is active)
6. **Finish**: Summary and next steps

## Homepage Setup

AquaLuxe includes a pre-designed homepage that you can customize to fit your needs.

### Using the Pre-designed Homepage
1. Go to **Pages > Add New**
2. Enter a title (e.g., "Home")
3. Select the **Homepage** template from the Page Attributes section
4. Click **Publish**
5. Go to **Settings > Reading**
6. Select **A static page** for "Your homepage displays"
7. Choose your newly created page for "Homepage"
8. Click **Save Changes**

### Customizing the Homepage
1. Edit the homepage you created
2. If using Elementor:
   - Click **Edit with Elementor**
   - Use the drag-and-drop interface to customize the page
3. If using the default editor:
   - Use the block editor to add and arrange content
   - Add sections using the available blocks

### Homepage Sections
The pre-designed homepage includes the following sections:
- Hero slider
- Featured fish species
- About section
- Featured products
- Testimonials
- Latest care guides
- Newsletter signup
- Instagram feed

## Navigation Menus

AquaLuxe supports multiple navigation menus:

### Available Menu Locations
- **Primary Menu**: Main navigation in the header
- **Footer Menu**: Links in the footer
- **Social Links Menu**: Social media icons
- **Mobile Menu**: Navigation for mobile devices

### Creating and Assigning Menus
1. Go to **Appearance > Menus**
2. Click **Create a new menu**
3. Enter a menu name
4. Add pages, categories, custom links, or other items to the menu
5. Under "Menu Settings," select the display location(s)
6. Click **Save Menu**

### Adding Custom Links
1. In the menu editor, expand the **Custom Links** section
2. Enter the URL and link text
3. Click **Add to Menu**
4. Arrange the link in the desired position
5. Click **Save Menu**

### Creating Dropdown Menus
1. In the menu editor, add the parent and child menu items
2. Drag the child items slightly to the right under the parent item
3. The child items will be indented to show they are submenu items
4. Click **Save Menu**

## Custom Post Types

AquaLuxe includes several custom post types for specific content:

### Fish Species
Use this post type to showcase different fish species you offer:

1. Go to **Fish Species > Add New**
2. Enter the fish species name as the title
3. Add a description in the main content area
4. Set the following information in the custom fields:
   - **Scientific Name**: Latin name of the species
   - **Adult Size**: Maximum size of the adult fish
   - **Lifespan**: Average lifespan
   - **pH Range**: Suitable pH range
   - **Temperature**: Suitable temperature range
   - **Tank Size**: Minimum tank size
5. Select taxonomies:
   - **Fish Family**: Taxonomic family
   - **Fish Origin**: Geographic origin
   - **Water Type**: Freshwater, saltwater, brackish
   - **Care Level**: Beginner, intermediate, expert
6. Add a featured image
7. Click **Publish**

### Care Guides
Use this post type to create care guides for fish keeping:

1. Go to **Care Guides > Add New**
2. Enter the guide title
3. Add the guide content
4. Set the following information in the custom fields:
   - **Difficulty Level**: Easy, medium, hard, expert
   - **Time Required**: Estimated time to complete
   - **Materials Needed**: List of required materials
   - **Guide Steps**: Step-by-step instructions
5. Select a **Guide Category**
6. Add a featured image
7. Click **Publish**

### Team Members
Use this post type to showcase your team:

1. Go to **Team > Add New**
2. Enter the team member's name
3. Add their bio in the content area
4. Set the following information in the custom fields:
   - **Position**: Job title
   - **Email**: Contact email
   - **Phone**: Contact phone
   - **Social Links**: Social media profiles
   - **Expertise**: Areas of expertise
5. Add a profile photo as the featured image
6. Click **Publish**

### Testimonials
Use this post type to display customer testimonials:

1. Go to **Testimonials > Add New**
2. Enter the testimonial title
3. Add the testimonial text in the content area
4. Set the following information in the custom fields:
   - **Client Name**: Name of the person giving the testimonial
   - **Client Position**: Job title or company
   - **Rating**: Star rating (1-5)
   - **Date**: Date of the testimonial
5. Add a client photo as the featured image (optional)
6. Click **Publish**

### FAQs
Use this post type to create a frequently asked questions section:

1. Go to **FAQs > Add New**
2. Enter the question as the title
3. Add the answer in the content area
4. Select an **FAQ Category**
5. Click **Publish**

## WooCommerce Integration

AquaLuxe is fully integrated with WooCommerce for selling products online:

### Setting Up Your Store
1. Make sure WooCommerce is installed and activated
2. Go to **WooCommerce > Setup Wizard** to configure basic store settings
3. Follow the wizard to set up:
   - Store location
   - Currency
   - Shipping
   - Payment methods
   - Tax settings

### Creating Products
1. Go to **Products > Add New**
2. Enter the product name
3. Add the product description
4. Set the product data:
   - **Product Type**: Simple, variable, grouped, etc.
   - **Price**: Regular and sale prices
   - **Inventory**: SKU, stock status, etc.
   - **Shipping**: Weight, dimensions, shipping class
   - **Linked Products**: Up-sells, cross-sells
   - **Attributes**: Size, color, etc.
   - **Advanced**: Purchase note, menu order, etc.
5. Add product categories and tags
6. Add product images
7. Click **Publish**

### Customizing Shop Pages
1. Go to **Appearance > Customize**
2. Click on **WooCommerce**
3. Use the various sections to customize:
   - **Product Catalog**: Layout, columns, etc.
   - **Product Images**: Size, zoom, lightbox
   - **Checkout**: Fields, options, etc.

### Advanced Product Features

#### Quick View
AquaLuxe includes a quick view feature that allows customers to preview products without leaving the shop page:

1. Go to **AquaLuxe > Theme Options**
2. Click on **WooCommerce**
3. Enable **Quick View**
4. Configure quick view options:
   - **Show Price**: Yes/No
   - **Show Rating**: Yes/No
   - **Show Add to Cart**: Yes/No
   - **Show Description**: Yes/No

#### Wishlist
The wishlist feature allows customers to save products for later:

1. Go to **AquaLuxe > Theme Options**
2. Click on **WooCommerce**
3. Enable **Wishlist**
4. Configure wishlist options:
   - **Require Login**: Yes/No
   - **Show in Menu**: Yes/No
   - **Show on Product**: Yes/No

#### Advanced Filtering
AquaLuxe includes advanced product filtering:

1. Go to **Appearance > Widgets**
2. Add filter widgets to the Shop Sidebar:
   - **Filter Products by Price**
   - **Filter Products by Attribute**
   - **Active Product Filters**
3. Configure each widget as needed

## Multilingual Support

AquaLuxe is compatible with popular translation plugins like WPML and Polylang:

### Setting Up WPML
1. Install and activate WPML plugins:
   - WPML Multilingual CMS
   - WPML String Translation
   - WPML Translation Management
2. Go to **WPML > Languages**
3. Add your languages
4. Configure language settings
5. Go to **WPML > Theme and Plugin Localization**
6. Scan the theme for strings
7. Translate theme strings in **WPML > String Translation**

### Setting Up Polylang
1. Install and activate Polylang
2. Go to **Languages > Settings**
3. Add your languages
4. Configure language settings
5. Go to **Languages > Strings Translations**
6. Translate theme strings

### Translating Content
1. Create or edit a post/page
2. Look for the language metabox
3. Click the "+" icon next to the language you want to translate to
4. Create the translation
5. Click **Publish**

### Language Switcher
AquaLuxe includes a language switcher in the header:

1. Go to **Appearance > Customize**
2. Click on **Header**
3. Enable **Language Switcher**
4. Configure language switcher options:
   - **Display Style**: Text, Flags, Both
   - **Show Language Names**: Yes/No
   - **Show in Mobile Menu**: Yes/No

## Performance Features

AquaLuxe includes several performance optimization features:

### Image Optimization
1. Go to **Settings > Media**
2. Scroll to the **AquaLuxe Image Optimization** section
3. Configure the following options:
   - **Enable WebP Conversion**: Convert JPEG and PNG images to WebP format
   - **WebP Quality**: Set the quality of WebP images
   - **Enable Image Compression**: Compress uploaded images
   - **Compression Quality**: Set the quality of compressed images

### Caching
1. Go to **Settings > General**
2. Scroll to the **AquaLuxe Caching** section
3. Configure the following options:
   - **Enable Browser Caching**: Add browser caching headers
   - **Enable Page Caching**: Cache pages for faster loading
   - **Enable Asset Optimization**: Optimize asset loading
   - **Remove Query Strings**: Remove query strings from static resources
   - **Enable REST API Caching**: Add cache headers to REST API responses

### Purging Cache
To purge the cache:
1. In the admin bar, click on **Cache**
2. Select **Purge Current Page Cache** or **Purge All Cache**

## Troubleshooting

### Common Issues

#### 1. Theme Not Displaying Correctly
- Make sure you have the latest version of WordPress and the theme
- Check if all required plugins are installed and activated
- Try switching to a default WordPress theme to see if the issue persists
- Check your browser console for JavaScript errors

#### 2. WooCommerce Pages Not Styled
- Make sure WooCommerce is installed and activated
- Go to **WooCommerce > Status > Tools** and click **Regenerate product lookup tables**
- Go to **WooCommerce > Status > Tools** and click **Regenerate product attributes lookup table**

#### 3. Custom Post Types Not Showing
- Go to **Settings > Permalinks** and click **Save Changes** to refresh permalinks
- Check if the custom post type is enabled in the theme options

#### 4. Images Not Loading
- Check if the image files exist in the uploads directory
- Try regenerating thumbnails using a plugin like "Regenerate Thumbnails"
- Check if your server supports WebP images (if WebP conversion is enabled)

#### 5. Performance Issues
- Enable all caching options
- Optimize images
- Use a caching plugin like WP Rocket or W3 Total Cache
- Use a CDN service
- Contact your hosting provider for server optimization

### Resetting Theme Options
If you want to reset all theme options to default:
1. Go to **AquaLuxe > Theme Options**
2. Click on **Tools**
3. Click **Reset Options**
4. Confirm the reset
