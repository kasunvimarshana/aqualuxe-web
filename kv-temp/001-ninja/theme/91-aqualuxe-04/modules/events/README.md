# Events Module

This module provides integration and custom features for [The Events Calendar](https://wordpress.org/plugins/the-events-calendar/) plugin.

## Features

*   Provides a basic structure for adding custom styles for the events calendar and single event pages.

## Setup

1.  **Install and activate The Events Calendar plugin.**
2.  **Create Events:** Add your events in the "Events" section of the WordPress admin.

## Template Overrides

To customize the appearance of the events calendar and single event pages, you can override the default templates. Copy the template files from the plugin's `src/views` directory to the `aqualuxe/tribe-events/` directory in your theme and then customize them.

For more details, see The Events Calendar's documentation on template overrides.
