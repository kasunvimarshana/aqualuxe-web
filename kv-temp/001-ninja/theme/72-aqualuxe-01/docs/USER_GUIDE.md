# AquaLuxe User Guide

- Customize: Appearance > Customize (colors, typography, layout)
- Menus: Appearance > Menus (assign to Primary, Footer)
- Pages: Home, About, Services, Blog, Contact, FAQ are created via Demo Importer
- Contact: Edit Contact page content and map; form uses built-in handler
- WooCommerce: Install/activate to enable shop features, quick view, filters, wishlist
- Dark Mode: Toggle moon icon in the header, persists per browser
- Modules: Toggle modules in each `modules/*/module.json` (enabled true/false)
