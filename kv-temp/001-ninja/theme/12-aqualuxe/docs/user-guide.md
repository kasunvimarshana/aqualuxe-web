## Installation

### Requirements

Before installing AquaLuxe, ensure your hosting environment meets these requirements:
- WordPress 5.6 or higher
- PHP 7.4 or higher
- MySQL 5.6 or higher
- WooCommerce 5.0 or higher (if using e-commerce features)

### Installing the Theme

1. Log in to your WordPress admin panel
2. Go to **Appearance > Themes**
3. Click **Add New** and then **Upload Theme**
4. Choose the AquaLuxe zip file and click **Install Now**
5. After installation, click **Activate** to activate the theme

### Installing Required Plugins

AquaLuxe works best with the following plugins:

1. **WooCommerce**: For e-commerce functionality
2. **Contact Form 7**: For contact forms
3. **Advanced Custom Fields**: For custom fields
4. **Elementor**: For page building (optional)

After activating the theme, you'll see a notification recommending these plugins. Click **Begin installing plugins** to install them.

## Theme Setup

### Initial Setup Wizard

AquaLuxe includes a setup wizard to help you configure the theme:

1. After activating the theme, you'll see a welcome message
2. Click **Run the Setup Wizard**
3. Follow the steps to:
   - Import demo content (optional)
   - Set up your logo and site identity
   - Configure basic theme options
   - Set up WooCommerce (if installed)

### Manual Setup

If you prefer to set up the theme manually:

1. Go to **Appearance > Customize**
2. Configure the following sections:
   - **Site Identity**: Upload your logo, set site title and tagline
   - **Colors**: Set your primary and secondary colors
   - **Typography**: Choose fonts for headings and body text
   - **Header**: Configure header layout and options
   - **Footer**: Set up footer widgets and copyright text
   - **Homepage**: Configure homepage sections
   - **Blog**: Set blog layout and options
   - **WooCommerce**: Configure shop settings (if WooCommerce is active)

## Customizing Your Site

### Theme Customizer

The WordPress Customizer is the main tool for customizing your site:

1. Go to **Appearance > Customize**
2. Make changes in the various sections
3. Preview changes in real-time
4. Click **Publish** to save your changes

### Site Identity

1. Go to **Appearance > Customize > Site Identity**
2. Upload your logo (recommended size: 180px × 60px)
3. Set your site title and tagline
4. Upload a site icon (favicon)

### Colors

1. Go to **Appearance > Customize > Colors**
2. Set your primary color (used for buttons, links, and accents)
3. Set your secondary color
4. Configure additional color options for various elements

### Typography

1. Go to **Appearance > Customize > Typography**
2. Choose fonts for headings and body text
3. Set font sizes for different elements
4. Configure font weights and line heights

### Header

1. Go to **Appearance > Customize > Header**
2. Choose a header layout
3. Configure top bar options (contact info, social icons)
4. Set up navigation menu options
5. Configure sticky header behavior

### Footer

1. Go to **Appearance > Customize > Footer**
2. Set the number of footer widget columns
3. Configure newsletter signup options
4. Set copyright text
5. Configure payment icons and social links

## Homepage Setup

### Using the Homepage Template

1. Create a new page by going to **Pages > Add New**
2. Enter a title (e.g., "Home")
3. Select **Homepage** from the **Template** dropdown in the **Page Attributes** section
4. Publish the page
5. Go to **Settings > Reading**
6. Set **Your homepage displays** to **A static page**
7. Select your new page as the **Homepage**
8. Click **Save Changes**

### Configuring Homepage Sections

1. Go to **Appearance > Customize > Homepage**
2. Configure each section:

   #### Hero Slider
   - Enable/disable the slider
   - Add slides with images, titles, text, and buttons
   - Configure slider options (autoplay, speed, etc.)

   #### Features Section
   - Enable/disable the section
   - Set section title and subtitle
   - Add feature items with icons, titles, and text

   #### About Section
   - Enable/disable the section
   - Set section title, subtitle, and content
   - Upload an image
   - Configure button text and link

   #### Species Showcase
   - Enable/disable the section
   - Set section title and subtitle
   - Choose number of species to display
   - Configure "View All" button

   #### Featured Products
   - Enable/disable the section
   - Set section title and subtitle
   - Choose number of products to display
   - Configure "Shop Now" button

   #### Testimonials
   - Enable/disable the section
   - Set section title and subtitle
   - Choose number of testimonials to display

   #### Team Members
   - Enable/disable the section
   - Set section title and subtitle
   - Choose team members to display

   #### Blog Posts
   - Enable/disable the section
   - Set section title and subtitle
   - Choose number of posts to display
   - Configure "View All" button

   #### Call to Action
   - Enable/disable the section
   - Set title, text, button text, and link
   - Upload background image

## Working with Species

AquaLuxe includes a custom post type for showcasing aquatic species.

### Adding a New Species

1. Go to **Species > Add New**
2. Enter the species name as the title
3. Add description in the main content area
4. Set featured image (main species photo)
5. Fill in species details in the **Species Details** meta box:
   - Scientific Name
   - Origin
   - Habitat
   - Care Level
   - Size
   - Lifespan
   - Diet
   - Temperature
   - pH Level
   - Tank Size
   - Temperament
   - Breeding Information
6. Add gallery images in the **Species Gallery** section
7. Add a video URL in the **Species Video** field
8. Assign species categories and tags
9. Click **Publish**

### Managing Species Categories

1. Go to **Species > Categories**
2. Add new categories as needed
3. You can create a hierarchy of categories (e.g., Freshwater > Tetras)

### Species Archive Page

The species archive page displays all your species in a grid layout:

1. Go to **Appearance > Customize > Species**
2. Configure archive page options:
   - Archive title and subtitle
   - Number of species per page
   - Grid columns
   - Filtering options

## Working with WooCommerce

AquaLuxe is fully compatible with WooCommerce for selling aquatic products.

### Setting Up WooCommerce

1. Install and activate WooCommerce
2. Follow the WooCommerce setup wizard
3. Go to **Appearance > Customize > WooCommerce** to configure theme-specific options

### Product Management

1. Go to **Products > Add New** to add a new product
2. Enter product details (name, description, price, etc.)
3. Set product categories and tags
4. Upload product images
5. Configure product data (shipping, inventory, etc.)
6. For aquatic species products, use the **Species Information** tab to link to a species post

### Shop Page Customization

1. Go to **Appearance > Customize > WooCommerce > Product Catalog**
2. Configure shop page options:
   - Products per page
   - Columns
   - Product sorting
   - Sale badge style
   - Quick view options

### Cart and Checkout

1. Go to **Appearance > Customize > WooCommerce > Cart & Checkout**
2. Configure cart and checkout options:
   - Cart layout
   - Cross-sells display
   - Checkout fields
   - Order notes

## Blog Management

   - Blog layout (grid or list)

   - Featured image size

2. Enter post title and content

4. Assign categories and tags
5. Configure excerpt (optional)
6. Set publish date and status
7. Click **Publish**

## Troubleshooting

### Common Issues

1. **Images not displaying**: Ensure you've uploaded images with the correct dimensions
2. **Layout issues**: Check if you're using the latest version of the theme and compatible plugins
3. **Customizer changes not saving**: Clear your browser cache or try a different browser
4. **WooCommerce styling issues**: Make sure WooCommerce is updated to the latest version

### Performance Optimization

1. Use a caching plugin like WP Super Cache or W3 Total Cache
2. Optimize images before uploading
3. Use a CDN for assets
4. Choose a quality hosting provider
