# AquaLuxe WordPress Theme

AquaLuxe is a premium WordPress theme designed specifically for high-end ornamental fish farming businesses, aquarium shops, and aquatic specialists. The theme features full WooCommerce integration, elegant design, and a responsive layout to showcase aquatic species and products beautifully.

## Features

### General Features
- Fully responsive design that looks great on all devices
- Modern and clean aesthetic with an aquatic-inspired color scheme
- Customizable homepage with multiple sections
- Custom post types for Species, Team Members, Testimonials, FAQs, and Gallery
- Advanced theme customizer options
- SEO optimized structure
- Translation ready
- Child theme compatible
- Regular updates

### WooCommerce Integration
- Fully styled shop pages
- Custom product display for aquatic species
- AJAX add-to-cart functionality
- Quick view product modals
- Wishlist functionality
- Product comparison
- Advanced filtering options
- Custom checkout fields for aquatic products

### Performance
- Optimized code for fast loading
- Minified CSS and JavaScript
- Lazy loading for images
- Responsive images for different screen sizes

## Requirements
- WordPress 5.6 or higher
- PHP 7.4 or higher
- WooCommerce 5.0 or higher (if using e-commerce features)

## Installation

### Manual Installation
1. Download the theme zip file
2. Log in to your WordPress admin panel
3. Go to Appearance > Themes
4. Click "Add New" and then "Upload Theme"
5. Choose the downloaded zip file and click "Install Now"
6. After installation, click "Activate" to activate the theme

### Required Plugins
For full functionality, the following plugins are recommended:
- WooCommerce (for e-commerce functionality)
- Contact Form 7 (for contact forms)
- Advanced Custom Fields (for custom fields)
- Elementor (for page building)

## Customization

### Theme Customizer
The theme includes extensive customization options through the WordPress Customizer:
1. Go to Appearance > Customize
2. Explore the various sections to customize:
   - Site Identity
   - Colors
   - Typography
   - Header Options
   - Footer Options
   - Homepage Sections
   - Blog Settings
   - WooCommerce Settings
   - Species Settings
   - And more...

### Custom Post Types
The theme includes the following custom post types:
- **Species**: For showcasing different fish and aquatic species
- **Team**: For displaying team members/staff
- **Testimonials**: For customer reviews and testimonials
- **FAQs**: For frequently asked questions
- **Gallery**: For photo galleries

### Page Templates
The theme includes several custom page templates:
- **Homepage**: A custom homepage with multiple sections
- **About**: A template for about/company information
- **Contact**: A template with contact form and Google Maps
- **FAQ**: A template for displaying FAQs
- **Gallery**: A template for displaying photo galleries
- **Full Width**: A template without sidebar
