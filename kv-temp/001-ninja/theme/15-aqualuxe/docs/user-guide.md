## Installation <a name="installation"></a>

### Requirements
- WordPress 5.6 or higher
- PHP 7.4 or higher
- MySQL 5.6 or higher
- WooCommerce 5.0 or higher (if using e-commerce features)

### Installation Steps

1. **Upload the Theme**
   - Go to your WordPress admin panel
   - Navigate to Appearance > Themes
   - Click "Add New"
   - Click "Upload Theme"
   - Choose the aqualuxe.zip file
   - Click "Install Now"
   - After installation, click "Activate"

2. **Install Required Plugins**
   - After activating the theme, you'll see a notification about recommended plugins
   - Click "Begin installing plugins"
   - Select all recommended plugins
   - Click "Install"
   - After installation, activate all plugins

3. **Import Demo Content (Optional)**
   - Go to Tools > Import
   - Click on "WordPress" (install the importer if prompted)
   - Choose the demo-content.xml file from the theme package
   - Click "Upload file and import"
   - Assign authors as prompted
   - Check "Download and import file attachments"
   - Click "Submit"

## Theme Setup <a name="theme-setup"></a>

### Initial Setup Checklist

1. **Set Up Your Logo**
   - Go to Appearance > Customize
   - Click on "Site Identity"
   - Upload your logo
   - Adjust the logo size if needed
   - Set your site title and tagline
   - Save changes

2. **Configure Menus**
   - Go to Appearance > Menus
   - Create a new menu or edit existing ones
   - Add pages, categories, custom links, etc.
   - Assign the menu to a location (Primary Menu, Footer Menu, etc.)
   - Save the menu

3. **Set Up Homepage**
   - Go to Pages > Add New
   - Create a new page titled "Home"
   - Select the "Front Page" template
   - Add your content using the block editor
   - Publish the page
   - Go to Settings > Reading
   - Set "Your homepage displays" to "A static page"
   - Select your new "Home" page as the homepage
   - Save changes

4. **Configure Widgets**
   - Go to Appearance > Widgets
   - Add widgets to the available widget areas (Sidebar, Footer 1-4, etc.)
   - Customize widget settings as needed
   - Save changes

5. **Set Up WooCommerce (if using)**
   - Go to WooCommerce > Settings
   - Follow the setup wizard to configure your store
   - Set up payment methods, shipping, taxes, etc.
   - Add products to your store

## Working with WooCommerce <a name="working-with-woocommerce"></a>

AquaLuxe is fully compatible with WooCommerce and includes custom styling and enhanced functionality.

### Setting Up Your Shop

1. **Configure WooCommerce Settings**
   - Go to WooCommerce > Settings
   - Set up payment methods
   - Configure shipping options
   - Set tax rates
   - Customize email templates

2. **Create Product Categories**
   - Go to Products > Categories
   - Add categories relevant to your aquarium business (e.g., Freshwater Fish, Saltwater Fish, Plants, Equipment)
   - Add descriptions and images for each category

3. **Add Products**
   - Go to Products > Add New
   - Enter product title, description, and short description
   - Set product data (price, inventory, shipping, etc.)
   - Add product images
   - Assign categories and tags
   - Publish the product

### Shop Customization

1. **Shop Page Layout**
   - Go to Appearance > Customize > WooCommerce > Product Catalog
   - Set the number of products per row
   - Choose the shop page layout (grid or list)
   - Set the number of products per page

2. **Single Product Page**
   - Go to Appearance > Customize > WooCommerce > Single Product
   - Enable/disable related products
   - Set the number of related products
   - Configure product image zoom
   - Enable/disable product tabs

3. **Cart and Checkout**
   - Go to Appearance > Customize > WooCommerce > Cart & Checkout
   - Enable/disable cross-sells
   - Set the number of cross-sell products
   - Configure multi-step checkout
   - Customize cart and checkout fields

## Custom Post Types <a name="custom-post-types"></a>

AquaLuxe includes three custom post types specifically designed for aquarium businesses:

### Services

The Services post type allows you to showcase the services you offer, such as aquarium maintenance, custom tank design, or fish health consultations.

**Adding a Service:**
1. Go to Services > Add New
2. Enter the service title and description
3. Set the service price and duration (using custom fields)
4. Add a featured image
5. Assign service categories
6. Publish the service

**Displaying Services:**
- Use the Services page template to display all services
- Add the Services shortcode to any page: `[aqualuxe_services]`
- Use the Services widget in widget areas

### Events

The Events post type allows you to promote upcoming events, workshops, or exhibitions related to your aquarium business.

**Adding an Event:**
1. Go to Events > Add New
2. Enter the event title and description
3. Set the event date, time, and location (using custom fields)
4. Add a featured image
5. Assign event types
6. Publish the event

**Displaying Events:**
- Events are automatically displayed in chronological order
- Use the Events shortcode to display events: `[aqualuxe_events]`
- Use the Events widget to display upcoming events

### Testimonials

The Testimonials post type allows you to showcase customer feedback and reviews.

**Adding a Testimonial:**
1. Go to Testimonials > Add New
2. Enter the testimonial content
3. Set the customer name, position, and company (using custom fields)
4. Add a customer photo as the featured image
5. Assign a rating (using custom fields)
6. Publish the testimonial

**Displaying Testimonials:**
- Use the Testimonials shortcode to display testimonials: `[aqualuxe_testimonials]`
- Use the Testimonials widget in widget areas
- Testimonials can be added to the homepage and about page templates

## Performance Features <a name="performance-features"></a>

AquaLuxe includes several performance optimization features to ensure fast loading times.

### Performance Features

- **Optimized Asset Loading**: CSS and JavaScript files are loaded conditionally
- **Lazy Loading**: Images and iframes are loaded only when needed
- **Critical CSS**: Above-the-fold CSS is loaded inline
- **Script Optimization**: Scripts are loaded with defer or async attributes
- **Resource Hints**: Preconnect and preload hints improve resource loading
- **Cache Busting**: Asset versions are based on file modification time

### Performance Settings

1. Go to Appearance > Customize > Performance
2. Enable/disable lazy loading
3. Configure asset loading
4. Enable/disable critical CSS
5. Set preload resources
6. Save changes

## Frequently Asked Questions <a name="frequently-asked-questions"></a>

### General Questions

**Q: How do I update the theme?**
A: When a new version is available, go to Appearance > Themes, click on the theme, and you'll see an update notification if available. Click "Update" to install the latest version.

**Q: Can I use AquaLuxe without WooCommerce?**
A: Yes, WooCommerce is optional. If you don't need e-commerce functionality, you can simply not install WooCommerce.

**Q: How do I change the color scheme?**
A: Go to Appearance > Customize > Colors and adjust the color settings to your preference.

**Q: Can I add custom CSS?**
A: Yes, go to Appearance > Customize > Additional CSS to add your custom CSS code.

### WooCommerce Questions

**Q: How do I set up shipping for aquarium products?**
A: Go to WooCommerce > Settings > Shipping and configure shipping zones, methods, and classes. You can set up special shipping rules for live fish and delicate aquarium items.

**Q: Can I offer local pickup for aquarium products?**
A: Yes, go to WooCommerce > Settings > Shipping, edit a shipping zone, and add the "Local Pickup" shipping method.

**Q: How do I set up different prices for wholesale customers?**
A: You can use a WooCommerce wholesale plugin like "Wholesale Prices for WooCommerce" to set up different pricing tiers.

### Technical Questions

**Q: What PHP version is required?**
A: AquaLuxe requires PHP 7.4 or higher.

**Q: Is AquaLuxe compatible with Gutenberg?**
A: Yes, AquaLuxe is fully compatible with the Gutenberg block editor.

**Q: Can I use AquaLuxe with page builders?**
A: Yes, AquaLuxe works well with popular page builders like Elementor, Beaver Builder, and Divi.
