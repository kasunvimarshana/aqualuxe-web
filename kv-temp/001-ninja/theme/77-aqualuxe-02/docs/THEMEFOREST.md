# ThemeForest Submission Notes

- Modular architecture (core vs modules) with toggleable features
- No CDN, hashed assets with mix-manifest.json
- WooCommerce-compatible and graceful degradation when inactive
- Accessibility: semantic HTML, aria labels, keyboard-friendly UI
- Security: nonces; sanitized inputs; escaped outputs; headers
- Performance: Tailwind + tree-shaken build; no render-blocking raw files
- Documentation: README, SETUP, LICENSE, CI workflow
- Demonstration content via in-theme importer
