# AquaLuxe Demo Content Importer

This documentation provides information about the AquaLuxe theme's demo content importer, including how to use it, what content it imports, and how to customize the imported content.

## Overview

The AquaLuxe Demo Content Importer allows you to quickly set up your website with sample content that showcases the theme's features and capabilities. The importer creates sample content for all custom post types included with the theme, sets up WooCommerce products, and configures theme settings.

## How to Use the Importer

1. Navigate to **Appearance > Import Demo Content** in your WordPress admin dashboard.
2. Select the content types you want to import by checking the corresponding checkboxes.
3. Click the **Import Demo Content** button to start the import process.
4. Wait for the import process to complete. This may take several minutes depending on your server speed and the amount of content being imported.
5. Once the import is complete, you can view and edit the imported content in the WordPress admin dashboard.

## Content Types

The following content types can be imported:

- **Blog Posts**: Sample blog posts with featured images
- **Pages**: Sample pages including Home, About, Contact, etc.
- **Services**: Sample aquarium services
- **Team Members**: Sample team member profiles
- **Testimonials**: Sample customer testimonials
- **Projects**: Sample aquarium installation projects
- **FAQs**: Sample frequently asked questions
- **Care Guides**: Sample fish care guides
- **Auctions**: Sample fish auctions
- **WooCommerce Products**: Sample fish and aquarium products
- **Theme Settings**: Theme customizer settings
- **Widgets**: Widget configurations for sidebars and footer
- **Menus**: Navigation menu structure

## Customizing Imported Content

After importing the demo content, you should customize it to fit your specific needs:

1. **Edit Pages**: Update the content of imported pages to reflect your business information.
2. **Update Images**: Replace demo images with your own images.
3. **Modify Products**: Update product information, prices, and images.
4. **Adjust Theme Settings**: Fine-tune theme settings in the WordPress Customizer.
5. **Customize Menus**: Edit the imported menus to match your site structure.

## Important Notes

- The importer will not delete any existing content, but it may modify some settings.
- All demo images are for demonstration purposes only and should be replaced with your own images before publishing your site.
- The import process may take several minutes to complete, especially on slower servers.
- It is recommended to run the importer on a fresh WordPress installation to avoid conflicts with existing content.

## Troubleshooting

If you encounter any issues during the import process:

1. **Check Server Requirements**: Make sure your server meets the minimum requirements for WordPress and the AquaLuxe theme.
2. **Increase PHP Limits**: If the import fails, try increasing the PHP memory limit and execution time in your server configuration.
3. **Check Error Logs**: Check your server's error logs for any PHP errors that may be causing the import to fail.
4. **Contact Support**: If you continue to experience issues, contact our support team for assistance.

## Technical Details

The demo content importer uses WordPress's built-in import/export functionality and custom code to handle the import of theme-specific content and settings. The importer is designed to be secure and efficient, with proper error handling and data validation.

All imported content is sanitized and validated to ensure it meets WordPress security standards. The importer also includes proper attribution for all demo images and content.
