## Installation

### Requirements

- WordPress 5.6 or higher
- PHP 7.4 or higher
- WooCommerce 5.0 or higher (optional, but recommended)

### Installation Steps

1. **Download the theme**:
   - Download the `aqualuxe.zip` file from your purchase location

2. **Install via WordPress Admin**:
   - Go to your WordPress admin panel
   - Navigate to Appearance > Themes
   - Click "Add New" and then "Upload Theme"
   - Choose the `aqualuxe.zip` file and click "Install Now"
   - After installation, click "Activate" to activate the theme

3. **Install Required Plugins**:
   - After activating the theme, you'll see a notification to install recommended plugins
   - Click "Begin installing plugins"
   - Select all recommended plugins and click "Install"
   - After installation, activate all plugins

## Theme Setup

### Initial Setup

1. **Import Demo Content** (optional):
   - Go to Tools > Import
   - Click on "WordPress" and install the WordPress importer if prompted
   - Upload the demo content XML file from the theme package
   - Follow the prompts to import the content

2. **Set Up Homepage**:
   - Go to Pages > Add New
   - Create a new page titled "Home"
   - Use the page builder or blocks to design your homepage
   - Go to Settings > Reading
   - Set "Your homepage displays" to "A static page"
   - Select your newly created "Home" page as the homepage

3. **Set Up Menu**:
   - Go to Appearance > Menus
   - Create a new menu
   - Add pages, categories, or custom links to your menu
   - Set the menu location to "Primary Menu"
   - Click "Save Menu"

## Customizer Options

AquaLuxe comes with extensive customization options. To access them:

1. Go to Appearance > Customize

### General Settings

- **Site Identity**: Upload your logo, set site title and tagline
- **Colors**: Set primary, secondary, and accent colors
- **Typography**: Choose fonts for headings and body text
- **Background**: Set background color or image

### Header Options

- **Header Layout**: Choose from standard, centered, split, or transparent layouts
- **Header Width**: Set header to contained or full width
- **Sticky Header**: Enable/disable sticky header
- **Transparent Header**: Enable/disable transparent header on specific pages
- **Header Elements**: Show/hide search, cart, account, and other elements

### Footer Options

- **Footer Layout**: Choose from various footer layouts
- **Footer Width**: Set footer to contained or full width
- **Footer Widgets**: Enable/disable footer widgets
- **Copyright Text**: Customize the copyright text
- **Footer Elements**: Show/hide social icons, payment methods, and other elements

### Layout Options

- **Container Width**: Set the maximum width for the content container
- **Sidebar Position**: Choose sidebar position for different page types
- **Content Layout**: Set content layout for different page types

### Blog Settings

- **Blog Layout**: Choose from standard, grid, masonry, or list layouts
- **Post Meta**: Show/hide author, date, categories, tags, and comments
- **Featured Image**: Show/hide featured image and set its size
- **Excerpt Length**: Set the number of words in excerpts
- **Read More Text**: Customize the "Read More" text

### WooCommerce Settings

- **Shop Layout**: Choose from grid, list, or masonry layouts
- **Product Cards**: Customize product card style
- **Product Page Layout**: Choose from standard, wide, gallery, or sticky layouts
- **Cart & Checkout**: Customize cart and checkout pages
- **Quick View**: Enable/disable quick view functionality
- **Wishlist**: Enable/disable wishlist functionality

### Performance Options

- **Lazy Loading**: Enable/disable lazy loading for images
- **Critical CSS**: Enable/disable critical CSS
- **Asset Optimization**: Enable/disable asset minification and combination
- **Caching**: Enable/disable browser caching
- **Service Worker**: Enable/disable service worker for offline support

### Advanced Settings

- **Custom CSS**: Add custom CSS code
- **Custom JavaScript**: Add custom JavaScript code
- **Header Code**: Add code to the head section
- **Footer Code**: Add code to the footer section

## Page Setup

### Creating Pages

1. Go to Pages > Add New
2. Enter a title for your page
3. Add content using the block editor or page builder
4. Set page attributes (template, parent, order)
5. Publish the page

### Page Templates

AquaLuxe includes several page templates:

- **Default**: Standard page layout with sidebar
- **Full Width**: Full width page without sidebar
- **Contained**: Contained width page without sidebar
- **Landing Page**: Full width page without header and footer
- **Contact**: Page template with contact form and map

To select a template:

1. Edit the page
2. In the Page Attributes section of the sidebar, select a template
3. Update the page

## WooCommerce Setup

### Store Setup

1. Go to WooCommerce > Settings
2. Configure general settings, products, tax, shipping, payments, and accounts
3. Set up shipping zones and methods
4. Set up payment methods

### Product Categories

1. Go to Products > Categories
2. Create categories for your products
3. Add name, slug, description, and optional image
4. Set parent category if needed
5. Click "Add New Category"

### Adding Products

1. Go to Products > Add New
2. Enter product title and description
3. Set product data (simple, variable, grouped, etc.)
4. Add product images and gallery
5. Set product categories and tags
6. Configure product attributes if needed
7. Set product price and inventory
8. Publish the product

### Shop Page Customization

1. Go to Appearance > Customize > WooCommerce
2. Customize shop page layout, product cards, and other settings
3. Save changes

## Menus and Navigation

### Creating Menus

1. Go to Appearance > Menus
2. Click "Create a new menu"
3. Enter a name for your menu
4. Add pages, categories, custom links, or other items
5. Organize menu items by dragging and dropping
6. Set menu location
7. Click "Save Menu"

### Menu Locations

AquaLuxe includes several menu locations:

- **Primary Menu**: Main navigation in the header
- **Mobile Menu**: Navigation for mobile devices
- **Footer Menu**: Navigation in the footer
- **Legal Menu**: For legal pages in the footer

### Mega Menu

To create a mega menu:

1. Install and activate the recommended mega menu plugin
2. Go to Appearance > Menus
3. Add items to your menu
4. Click the arrow on a top-level menu item
5. Enable the "Mega Menu" option
6. Configure columns and content
7. Save the menu

## Dark Mode

AquaLuxe includes a built-in dark mode feature:

### Enabling Dark Mode

1. Go to Appearance > Customize > General
2. Enable the "Dark Mode" option
3. Configure dark mode settings:
   - Default Mode: Light, Dark, or Auto (based on system preference)
   - Dark Mode Colors: Customize colors for dark mode
   - Dark Mode Toggle: Show/hide dark mode toggle button
4. Save changes

### Using Dark Mode

- Click the dark mode toggle button in the header to switch between light and dark modes
- The theme will remember your preference for future visits
- If set to "Auto", the theme will follow your system preference

## Performance Options

AquaLuxe includes various performance optimization features:

### Optimizing Performance

1. Go to Appearance > Customize > Performance
2. Configure performance settings:
   - Image Optimization: Enable/disable WebP conversion and lazy loading
   - Asset Optimization: Enable/disable minification and combination
   - Caching: Enable/disable browser caching
   - Critical CSS: Enable/disable critical CSS generation
   - Service Worker: Enable/disable service worker for offline support

### Regenerating Critical CSS

If you make significant changes to your site design:

1. Go to Appearance > AquaLuxe
2. Click the "Regenerate Critical CSS" button
3. Wait for the process to complete

## Troubleshooting

### Common Issues

#### Theme Not Working Properly

1. Make sure you have the latest version of WordPress and the theme
2. Check if all required plugins are installed and activated
3. Try deactivating all plugins to see if there's a conflict
4. Try switching to a default WordPress theme to see if the issue persists
5. Check your server's PHP version and make sure it meets the requirements

#### WooCommerce Issues

1. Make sure you have the latest version of WooCommerce
2. Go to WooCommerce > Status to check for any issues
3. Try resetting WooCommerce settings to default
4. Check if the issue persists with a default WordPress theme

#### Performance Issues

1. Check your hosting environment
2. Enable performance optimization features
3. Optimize images and reduce their size
4. Minimize the use of plugins
5. Use a caching plugin
