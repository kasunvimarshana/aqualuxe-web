## 1. Introduction

Welcome to AquaLuxe! This guide will walk you through installing, configuring, and using the AquaLuxe WordPress theme. AquaLuxe is a modern, modular, and highly customizable theme designed for businesses and individuals who want a professional and flexible online presence.

## 2. Theme Installation

1.  Download the `aqualuxe.zip` file from your purchase source.
2.  In your WordPress admin dashboard, navigate to **Appearance > Themes**.
3.  Click the **Add New** button at the top of the page.
4.  Click the **Upload Theme** button.
5.  Choose the `aqualuxe.zip` file you downloaded and click **Install Now**.
6.  Once the theme is installed, click **Activate**.

## 3. Required Plugins

For the theme to function correctly, you must install and activate the following plugins:

*   **WooCommerce**: Required for all e-commerce functionality.
*   **Aqualuxe Core**: A companion plugin for this theme that manages core features (if provided separately).

You may be prompted to install these plugins automatically after activating the theme.

## 4. Theme Configuration

All theme options can be configured through the WordPress Customizer.

### Customizer Settings

Navigate to **Appearance > Customize** to access the theme options. Here you will find sections for:

*   **Site Identity**: Set your site title, tagline, and logo.
*   **Colors**: Manage your site's color scheme.
*   **Menus**: Configure your site's navigation menus.
*   **Widgets**: Add widgets to your sidebar and footer.
*   **Homepage Settings**: Choose what to display on your homepage.
*   **Maintenance Mode**: Enable/disable maintenance mode and set a custom message.
*   **Social Sharing**: Enable/disable social sharing buttons on posts.
*   **SEO Settings**: Configure basic SEO meta tags for your homepage.

### Menus

1.  Go to **Appearance > Menus**.
2.  Create a new menu and add your desired pages.
3.  Assign the menu to one of the available theme locations, such as "Primary Menu".

## 5. Using the Modules

### Dark Mode

The dark mode toggle is available in the bottom-right corner of the screen. Simply click the button to switch between light and dark modes. Your preference will be saved for your next visit.

### Maintenance Mode

1.  Go to **Appearance > Customize > Maintenance Mode**.
2.  Check the "Enable Maintenance Mode" box.
3.  Customize the message that will be displayed to visitors.
4.  Click **Publish**.

*Note: Logged-in administrators will still be able to see the full site.*

### Social Sharing

1.  Go to **Appearance > Customize > Social Sharing**.
2.  Ensure the "Enable Social Sharing Buttons" box is checked.
3.  Sharing buttons will now automatically appear at the bottom of your single posts.

### SEO Optimization

1.  Go to **Appearance > Customize > SEO Settings**.
2.  Fill in the "Home Page Meta Description" and "Home Page Meta Keywords" fields.
3.  Click **Publish**.

For single posts and pages, the theme will automatically use the post's excerpt (or a trimmed version of the content) as the meta description and the post's tags as meta keywords.

## 6. Demo Content Importer

To make your site look like the theme demo, you can import the demo content.

1.  Navigate to **Appearance > Demo Importer**.
2.  Click the **Import Demo Content** button.
3.  Wait for the import process to complete. This may take a few minutes.

**Important**: Only run the demo importer on a fresh WordPress installation. Running it on an existing site may overwrite your content.

## 7. Troubleshooting

*   **Theme is broken after activation**: Ensure all required plugins are installed and activated.
*   **Demo import fails**: Make sure your server meets the minimum requirements for WordPress and has a stable connection to the internet. Check your PHP error logs for more details.

For further assistance, please contact our support team.
