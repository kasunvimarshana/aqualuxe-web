# AquaLuxe WordPress Theme

![AquaLuxe Theme](screenshot.png)

## Description

AquaLuxe is a premium WordPress/WooCommerce theme designed specifically for aquatic businesses with the tagline "Bringing elegance to aquatic life – globally." This comprehensive theme offers a modular, multitenant, multivendor, multilingual, multicurrency, mobile-first, and fully responsive solution with a dual-state architecture that works with or without WooCommerce.

## Features

### Core Features
- **Modular Architecture**: Flexible, component-based design for easy customization
- **Dual-State Architecture**: Functions perfectly with or without WooCommerce
- **Mobile-First Design**: Optimized for mobile devices with responsive layouts
- **Dark Mode**: Built-in dark mode toggle with persistent user preference
- **Performance Optimized**: Fast loading times with optimized assets and code
- **Accessibility Ready**: WCAG 2.1 compliant for maximum accessibility

### E-Commerce Features
- **WooCommerce Integration**: Seamless integration with WooCommerce
- **Multivendor Support**: Compatible with major multivendor plugins (WC Marketplace, Dokan, WC Vendors, WCFM)
- **Multicurrency Support**: Integrated with popular currency plugins (WPML WooCommerce Multilingual, WOOCS, Currency Switcher for WooCommerce)
- **Product Quick View**: View product details without leaving the page
- **Enhanced Product Gallery**: Zoom, lightbox, and slider functionality
- **Custom Cart & Checkout**: Optimized checkout process for better conversions

### Multilingual Features
- **WPML Compatible**: Full integration with WPML for multilingual sites
- **RTL Support**: Right-to-left language support for global audiences
- **Translation Ready**: All strings are translation-ready

### Design Features
- **Multiple Header Layouts**: Choose from various header styles
- **Multiple Footer Layouts**: Flexible footer configuration
- **Blog Layouts**: Grid, list, and masonry layout options
- **Advanced Typography**: Extensive font options and controls
- **Custom Color Schemes**: Easily customize colors to match your brand
- **Elementor Compatible**: Works seamlessly with Elementor page builder

## Requirements

- WordPress 5.8 or higher
- PHP 7.4 or higher
- MySQL 5.6 or higher
- WooCommerce 6.0 or higher (if using e-commerce features)

## Installation

1. Upload the `aqualuxe` folder to the `/wp-content/themes/` directory
2. Activate the theme through the 'Themes' menu in WordPress
3. Configure the theme settings via Appearance > Customize > AquaLuxe Theme Options
4. If using WooCommerce, install and activate the plugin

## Customization

### Using the Customizer
Most theme settings can be configured through the WordPress Customizer:
1. Go to Appearance > Customize
2. Navigate through the AquaLuxe Theme Options panels
3. Make your changes and save

### Using a Child Theme
For advanced customizations, we recommend using a child theme:
1. Create a new folder named `aqualuxe-child` in your themes directory
2. Create a `style.css` file with the appropriate theme headers
3. Create a `functions.php` file to enqueue the parent and child theme styles
4. Activate the child theme
