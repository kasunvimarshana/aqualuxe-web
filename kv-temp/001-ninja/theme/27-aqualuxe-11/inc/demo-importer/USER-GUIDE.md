# AquaLuxe Demo Content Importer - User Guide

This user guide provides step-by-step instructions for using the AquaLuxe theme's demo content importer.

## Getting Started

### Prerequisites

Before using the demo content importer, make sure you have:

1. Installed and activated the AquaLuxe theme
2. Installed and activated all required plugins (WooCommerce, etc.)
3. A fresh WordPress installation (recommended, but not required)

### Accessing the Importer

1. Log in to your WordPress admin dashboard
2. Navigate to **Appearance > Import Demo Content**
3. You will see the AquaLuxe Demo Content Importer page

## Using the Importer

### Step 1: Select Content to Import

On the importer page, you'll see a list of content types that can be imported. Each content type has a checkbox next to it:

- **Blog Posts**: Sample blog posts with featured images
- **Pages**: Sample pages including Home, About, Contact, etc.
- **Services**: Sample aquarium services
- **Team Members**: Sample team member profiles
- **Testimonials**: Sample customer testimonials
- **Projects**: Sample aquarium installation projects
- **FAQs**: Sample frequently asked questions
- **Care Guides**: Sample fish care guides
- **Auctions**: Sample fish auctions
- **WooCommerce Products**: Sample fish and aquarium products
- **Theme Settings**: Theme customizer settings
- **Widgets**: Widget configurations for sidebars and footer
- **Menus**: Navigation menu structure

Check the boxes for the content types you want to import. By default, all content types are selected.

### Step 2: Start the Import Process

1. After selecting the content types, click the **Import Demo Content** button
2. The import process will begin, and you'll see a progress bar indicating the status
3. Do not close the browser window or navigate away from the page during the import process
4. The import process may take several minutes depending on your server speed and the amount of content being imported

### Step 3: Review the Import Results

Once the import process is complete, you'll see a summary of the imported content:

- Number of posts, pages, and other content types imported
- Any errors or warnings that occurred during the import
- Links to view the imported content

### Step 4: Customize the Imported Content

After importing the demo content, you should customize it to fit your specific needs:

1. **Edit Pages**: Update the content of imported pages to reflect your business information
   - Navigate to **Pages** in the WordPress admin dashboard
   - Click on a page to edit its content

2. **Update Images**: Replace demo images with your own images
   - When editing a post or page, click on an image to replace it
   - Upload your own image from the media library

3. **Modify Products**: Update product information, prices, and images
   - Navigate to **Products** in the WordPress admin dashboard
   - Click on a product to edit its details

4. **Adjust Theme Settings**: Fine-tune theme settings in the WordPress Customizer
   - Navigate to **Appearance > Customize**
   - Modify settings such as colors, fonts, and layout options

5. **Customize Menus**: Edit the imported menus to match your site structure
   - Navigate to **Appearance > Menus**
   - Add, remove, or rearrange menu items

## Troubleshooting

### Common Issues

1. **Import Process Takes Too Long**
   - Solution: Increase PHP memory limit and execution time in your server configuration
   - Contact your hosting provider for assistance

2. **Images Not Importing**
   - Solution: Make sure your server has enough disk space
   - Check file permissions on the uploads directory

3. **WooCommerce Products Not Importing**
   - Solution: Make sure WooCommerce is installed and activated
   - Check if your server meets WooCommerce requirements

4. **Import Process Fails**
   - Solution: Check your server's error logs for PHP errors
   - Try importing fewer content types at once

## Best Practices

1. **Use a Fresh WordPress Installation**: For best results, use the importer on a fresh WordPress installation
2. **Back Up Your Site**: Always back up your site before importing demo content
3. **Customize Content**: Replace demo content with your own content before launching your site
4. **Replace Demo Images**: Replace all demo images with your own images to avoid copyright issues
5. **Test Functionality**: After importing, test all functionality to ensure everything works as expected
