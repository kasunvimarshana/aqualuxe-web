== Description ==

AquaLuxe is a premium WordPress theme designed specifically for luxury aquatic retail businesses. It features full WooCommerce integration, dark mode support, multilingual capabilities, and a range of customization options.

= Core Features =
* Responsive design with Tailwind CSS
* Dark mode toggle with persistent preferences
* Multilingual support
* Customizable color schemes
* Custom post types and taxonomies
* Widget areas for sidebar and footer
* Custom page templates

= WooCommerce Integration =
* Custom product templates
* Quick view modals
* Advanced AJAX product filtering
* Wishlist functionality
* Multi-currency support
* Optimized international shipping
* Enhanced checkout experience

= Performance Optimization =
* Lazy loading for images and iframes
* CSS and JavaScript minification
* Image optimization with WebP support
* Browser caching strategies
* Resource preloading and prefetching
* Schema.org markup for SEO
* Open Graph metadata for social sharing

= Accessibility Features =
* ARIA landmarks and roles
* Keyboard navigation support
* Focus management
* Screen reader text for icons
* Skip links
* Sufficient color contrast
* Semantic HTML5 structure

== Installation ==

1. Upload the `aqualuxe` folder to the `/wp-content/themes/` directory
2. Activate the theme through the 'Themes' menu in WordPress
3. Configure theme settings through the WordPress Customizer
4. Install and activate WooCommerce if you want to use the e-commerce features

For detailed installation instructions, see the included documentation.

== Frequently Asked Questions ==

= Does this theme require any plugins? =

While AquaLuxe works great on its own, it is designed to integrate seamlessly with WooCommerce for e-commerce functionality. We also recommend using Yoast SEO for enhanced SEO features and Contact Form 7 for contact forms.

= Can I use this theme for a non-aquatic business? =

Absolutely! While AquaLuxe was designed with aquatic businesses in mind, its clean design and extensive customization options make it suitable for any luxury retail business.

= Does this theme support RTL languages? =

Yes, AquaLuxe includes full RTL (Right-to-Left) language support.

= Is this theme translation-ready? =

Yes, AquaLuxe is fully translation-ready and includes multilingual support.

= Does this theme support dark mode? =

Yes, AquaLuxe includes a dark mode toggle with persistent preferences. It can also automatically switch to dark mode based on the user's system preferences.
